Java 教程是为希望使用 Java 编程语言创建应用程序的程序员编写的实用指南。它们包含数百个完整且可运行的示例，以及数十节课。相互关联的课程被组织成「学习路径（trails）」。

## 基础学习路径（Trails Covering the Basics）

这些学习路径以图书形式收录在 *The Java Tutorial, Sixth Edition* 中。购买此书请参见右侧的链接框。
- [[入门|入门（Getting Started）]] —— Java 技术简介，以及安装 Java 开发软件并用其创建一个简单程序的课程。
- [[学习 Java 语言|学习 Java 语言（Learning the Java Language）]] —— 描述 Java 编程语言核心概念与特性的课程。
- [[Java 核心类|Java 核心类（Essential Java Classes）]] —— 关于异常、基本输入/输出、并发、正则表达式以及平台环境的课程。
- [[集合|集合（Collections）]] —— 关于使用和扩展 Java 集合框架（Java Collections Framework）的课程。
- [[日期时间 API|日期时间 API（Date-Time APIs）]] —— 如何使用 java.time 相关页面编写日期与时间代码。
- [[部署|部署（Deployment）]] —— 如何使用 JAR 文件打包应用程序和小应用程序（applet），并通过 Java Web Start 和 Java 插件（Java Plug-in）进行部署。
- [[Java 编程语言认证备考|Java 编程语言认证备考（Preparation for Java Programming Language Certification）]] —— 可用的培训与教程资源列表。

## 专题学习路径与课程（Specialized Trails and Lessons）

这些学习路径与课程仅以网页形式提供。

- [[自定义网络|自定义网络（Custom Networking）]] —— Java 平台强大网络特性的入门介绍。
- [[扩展机制|扩展机制（The Extension Mechanism）]] —— 如何使自定义 API 可供运行在 Java 平台上的所有应用程序使用。
- [[全屏独占模式 API|全屏独占模式 API（Full-Screen Exclusive Mode API）]] —— 如何编写更充分利用用户图形硬件的应用程序。
- [[泛型|泛型（Generics）]] —— 一项对类型系统的增强，支持对各种类型的对象进行操作，同时提供编译时类型安全。请注意本课程面向高级用户。[[学习 Java 语言|Java 语言（Java Language）]]学习路径中包含一节适合初学者的 [泛型（Generics）](https://docs.oracle.com/javase/tutorial/java/generics/index.html)课程。
- [[国际化|国际化（Internationalization）]] —— 关于如何设计软件使其能够轻松适配（本地化，localized）各种语言和地区的入门介绍。
- [[JavaBeans]] —— Java 平台的组件技术。
- [[JAXB]] —— 介绍 XML 绑定的 Java 架构（Java Architecture for XML Binding，JAXB）技术。
- [[JAXP]] —— 介绍用于 XML 处理的 Java API（Java API for XML Processing，JAXP）技术。
- [[JDBC 数据库访问|JDBC 数据库访问（JDBC Database Access）]] —— 介绍一种用于实现 Java 应用程序与各类数据库及数据源之间连接的 API。
- [[JMX]] —— Java 管理扩展（Java Management Extensions）提供了一种以标准方式管理应用程序、设备和服务等资源的方法。
- [[JNDI]] —— Java 命名与目录接口（Java Naming and Directory Interface）支持访问如 DNS 和 LDAP 等命名与目录服务。
- [[反射|反射（Reflection）]] —— 一种用于表示（「反映」）当前 Java 虚拟机（Java Virtual Machine）中各类、接口和对象的 API。
- [[RMI]] —— 远程方法调用（Remote Method Invocation）API 允许一个对象调用运行在另一台 Java 虚拟机上的对象的方法。
- [[安全|安全（Security）]] —— 帮助保护应用程序免受恶意软件侵害的 Java 平台特性。
- [[套接字直接协议|套接字直接协议（Sockets Direct Protocol）]] —— 如何启用套接字直接协议以利用 InfiniBand。
- [[声音|声音（Sound）]] —— 一种用于从应用程序播放声音数据的 API。
- [[2D 图形|2D 图形（2D Graphics）]] —— 如何在应用程序中显示与打印 2D 图形。
