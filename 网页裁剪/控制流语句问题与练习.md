## 控制流语句问题与练习

## 问题

1. Java 编程语言所支持的最基本的控制流语句是 \_\_\_ 语句。
2. \_\_\_ 语句允许任意数量的可能执行路径。
3. \_\_\_ 语句类似于 `while` 语句，但在循环的 \_\_\_ 处对其表达式求值。
4. 如何使用 `for` 语句编写一个无限循环？
5. 如何使用 `while` 语句编写一个无限循环？

## 练习

1. 考虑下面的代码片段。
	```java
	if (aNumber >= 0)
	    if (aNumber == 0)
	        System.out.println("first string");
	else System.out.println("second string");
	System.out.println("third string");
	```
	1. 如果 `aNumber` 为 3，您认为这段代码会产生什么输出？
		2. 编写一个包含上面代码片段的测试程序；使 `aNumber` 为 3。该程序的输出是什么？它与您预测的一致吗？请解释输出为何如此；换句话说，该代码片段的控制流是怎样的？
		3. 仅使用空格和换行，重新格式化该代码片段，使控制流更易于理解。
		4. 使用花括号 { 和 }，进一步阐明这段代码。
