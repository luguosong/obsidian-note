## 全屏独占模式

使用 Microsoft DirectX API 的程序员可能已经熟悉全屏独占模式。其他程序员可能对这个概念有些陌生。无论哪种情况，全屏独占模式都是 J2SE™ 1.4 版本的一项强大特性，它允许程序员挂起窗口系统以便可以直接在屏幕上绘制。
