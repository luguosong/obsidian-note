## 字符和字符串小结

大多数时候，如果你使用的是单个字符值，你会使用基本类型 `char`。然而，有时你需要将 char 作为对象使用——例如，作为期望对象的方法参数。Java 编程语言为此提供了一个*包装类（wrapper class）*，将 `char`"包装"在 `Character` 对象中。一个 `Character` 类型的对象包含单个类型为 `char` 的字段。这个 [`Character`](https://docs.oracle.com/javase/8/docs/api/java/lang/Character.html) 类还提供了许多有用的类（即静态）方法用于操作字符。

字符串是字符序列，在 Java 编程中被广泛使用。在 Java 编程语言中，字符串是对象。[`String`](https://docs.oracle.com/javase/8/docs/api/java/lang/String.html) 类有 60 多个方法和 13 个构造方法。

最常见的是，你用如下语句创建字符串

```java
String s = "Hello world!";
```

而不是使用某个 `String` 构造方法。

`String` 类有许多用于查找和检索子字符串的方法；然后可以使用 `+` 拼接运算符轻松地将它们重新组合成新的字符串。

`String` 类还包括许多实用方法，其中有 `split()`、`toLowerCase()`、`toUpperCase()` 和 `valueOf()`。后一个方法在将用户输入的字符串转换为数字时不可或缺。`Number` 子类也有用于在字符串和数字之间相互转换的方法。

除了 `String` 类之外，还有一个 [`StringBuilder`](https://docs.oracle.com/javase/8/docs/api/java/lang/StringBuilder.html) 类。使用 `StringBuilder` 对象有时会比使用字符串更高效。`StringBuilder` 类提供了一些对字符串有用的方法，其中包括 `reverse()`。不过，总体而言，`String` 类拥有更多样化的方法。

字符串可以使用 `StringBuilder` 构造方法转换为字符串构建器。字符串构建器可以使用 `toString()` 方法转换为字符串。
