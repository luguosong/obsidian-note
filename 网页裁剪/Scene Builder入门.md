# 使用 Scene Builder 构建 JavaFX 应用程序

本章逐步演示如何使用 JavaFX Scene Builder 工具创建一个简单的问题跟踪(issue-tracking)应用程序。它向你展示如何快速构建 JavaFX 应用程序的用户界面(UI)，将其连接到 Java 源代码，并处理数据与用户界面之间的交互。

本文档包含以下主题：

- [为本教程做准备](http://www.oracle.com/pls/topic/prepare-for-tutorial.htm#CEGJBHHA)
- [打开示例 FXML 文件](http://www.oracle.com/pls/topic/open-sample-file.htm#BABBFEBD)
- [创建 FXML 文件和基础面板](http://www.oracle.com/pls/topic/create-base-panes.htm#CIHDHEFE)
- [将 GUI 绑定到应用程序逻辑](http://www.oracle.com/pls/topic/bind-ui-to-logic.htm#CHDDADJE)
- [添加列表和表格视图](http://www.oracle.com/pls/topic/add-list-table-views.htm#CIAICDGB)
- [创建详细信息部分](http://www.oracle.com/pls/topic/add-details.htm#BABFHBHI)
- [添加工具栏](http://www.oracle.com/pls/topic/add-toolbar.htm#BABHJBFD)
- [使用样式表并预览 UI](http://www.oracle.com/pls/topic/preview-stylesheet.htm#BABIDJEI)
- [编译并运行应用程序](http://www.oracle.com/pls/topic/compile-run.htm#BABEGIAA)
