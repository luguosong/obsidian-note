## 内部类示例

要查看内部类的实际使用，首先考虑一个数组。在下面的示例中，您创建一个数组，用整数值填充它，然后仅按升序输出数组中偶数索引处的值。

下面的 [`DataStructure.java`](https://docs.oracle.com/javase/tutorial/java/javaOO/examples/DataStructure.java) 示例由以下部分组成：

- `DataStructure` 外部类，它包含一个构造方法，用于创建一个 `DataStructure` 实例，其中保存了一个填充了连续整数值（0、1、2、3 等等）的数组；还有一个方法，用于打印数组中具有偶数索引值的元素。
- `EvenIterator` 内部类，它实现了 `DataStructureIterator` 接口，该接口扩展了 [`Iterator`](https://docs.oracle.com/javase/8/docs/api/java/util/Iterator.html)`<`[`Integer`](https://docs.oracle.com/javase/8/docs/api/java/lang/Integer.html)`>` 接口。迭代器(iterator)用于遍历数据结构，通常具有测试是否到达最后一个元素、获取当前元素以及移动到下一个元素的方法。
- 一个 `main` 方法，它实例化一个 `DataStructure` 对象（`ds`），然后调用 `printEven` 方法，打印数组 `arrayOfInts` 中具有偶数索引值的元素。

```sql
public class DataStructure {
    
    // Create an array
    private final static int SIZE = 15;
    private int[] arrayOfInts = new int[SIZE];
    
    public DataStructure() {
        // fill the array with ascending integer values
        for (int i = 0; i < SIZE; i++) {
            arrayOfInts[i] = i;
        }
    }
    
    public void printEven() {
        
        // Print out values of even indices of the array
        DataStructureIterator iterator = this.new EvenIterator();
        while (iterator.hasNext()) {
            System.out.print(iterator.next() + " ");
        }
        System.out.println();
    }
    
    interface DataStructureIterator extends java.util.Iterator<Integer> { } 

    // Inner class implements the DataStructureIterator interface,
    // which extends the Iterator<Integer> interface
    
    private class EvenIterator implements DataStructureIterator {
        
        // Start stepping through the array from the beginning
        private int nextIndex = 0;
        
        public boolean hasNext() {
            
            // Check if the current element is the last in the array
            return (nextIndex <= SIZE - 1);
        }        
        
        public Integer next() {
            
            // Record a value of an even index of the array
            Integer retValue = Integer.valueOf(arrayOfInts[nextIndex]);
            
            // Get the next even element
            nextIndex += 2;
            return retValue;
        }
    }
    
    public static void main(String s[]) {
        
        // Fill the array with integer values and print out only
        // values of even indices
        DataStructure ds = new DataStructure();
        ds.printEven();
    }
}
```

输出为：

```text
0 2 4 6 8 10 12 14
```

注意，`EvenIterator` 类直接引用了 `DataStructure` 对象的 `arrayOfInts` 实例变量。

您可以使用内部类来实现辅助类(helper class)，就像本示例中所展示的那样。要处理用户界面事件，您必须知道如何使用内部类，因为事件处理机制大量使用了它们。

## 局部类和匿名类

还有另外两种内部类。您可以在方法体内声明一个内部类，这些类被称为局部类(local class)。您也可以在方法体内声明一个不命名的内部类，这些类被称为匿名类(anonymous class)。

## 修饰符

您可以为内部类使用与外部类其它成员相同的修饰符。例如，您可以使用访问修饰符 `private`、`public` 和 `protected` 来限制对内部类的访问，就像您使用它们来限制对其它类成员的访问一样。
