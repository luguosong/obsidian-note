## 数字的问题与练习

## 问题

1. 使用 API 文档找出以下问题的答案：
	1. 你可以使用哪个 `Integer` 方法将一个 `int` 转换为以十六进制表示该数字的字符串？例如，哪个方法能将整数 65 转换为字符串 "41"？
		2. 你会使用哪个 `Integer` 方法将一个以 5 进制表示的字符串转换为等价的 `int`？例如，你如何将字符串 "230" 转换为整数值 65？写出你用来完成此任务的代码。
		3. 你可以使用哪个 Double 方法来检测一个浮点数是否具有特殊值"非数字（Not a Number, `NaN`）"？
2. 下面这个表达式的值是多少，为什么？
	```java
	Integer.valueOf(1).equals(Long.valueOf(1))
	```

## 练习

1. 修改 [`MaxVariablesDemo`](https://docs.oracle.com/javase/tutorial/java/data/QandE/MaxVariablesDemo.java)，使其显示最小值而非最大值。你可以删除所有与变量 `aChar` 和 `aBoolean` 相关的代码。输出是什么？
2. 创建一个程序，从命令行读取数量不定的整数参数并将它们相加。例如，假设你输入以下内容：
	```text
	java Adder 1 3 2 10
	```
	程序应显示 `16` 然后退出。如果用户只输入一个参数，程序应显示错误消息。你可以基于 [`ValueOfDemo`](https://docs.oracle.com/javase/tutorial/java/data/examples/ValueOfDemo.java) 来编写你的程序。
3. 创建一个与前一个类似的程序，但有如下区别：
	- 它读取浮点参数，而非整数参数。
		- 它显示参数的总和，小数点后恰好保留两位。
	例如，假设你输入以下内容：
	```text
	java FPAdder 1 1e2 3.0 4.754
	```
	程序将显示 `108.75`。根据你的语言环境，小数点可能是逗号（`,`）而非句点（`.`）。
