## 学习路径：学习 Java 语言

本学习路径涵盖 Java 编程语言编程的基础知识。

## 子页面

- [[面向对象编程概念|面向对象编程概念（Object-Oriented Programming Concepts）]] —— 教授面向对象编程背后的核心概念：对象、消息、类和继承，并展示这些概念如何转化为代码。如果您已经熟悉面向对象编程，可以跳过本课程。
- [[语言基础|语言基础（Language Basics）]] —— 描述语言的传统特性，包括变量、数组、数据类型、运算符和控制流程。
- [[类和对象|类和对象（Classes and Objects）]] —— 如何编写用于创建对象的类，以及如何创建和使用对象。
- [[注解|注解（Annotations）]] —— 一种元数据形式，为编译器提供信息；本课程描述在程序中何处以及如何有效地使用注解。
- [[接口和继承|接口和继承（Interfaces and Inheritance）]] —— 接口是什么、为什么要编写以及如何编写；如何从一个类派生出另一个类，即子类如何从超类继承字段和方法，所有类如何派生自 `Object` 类，以及如何修改子类从超类继承的方法。
- [[数字和字符串|数字和字符串（Numbers and Strings）]] —— 如何使用 `Number` 和 `String` 对象，以及如何为输出格式化数据。
- [[泛型|泛型（Generics）]] —— Java 编程语言的一项强大特性，能提高代码的类型安全，使更多的 bug 能够在编译时被检测出来。
- [[包|包（Packages）]] —— 帮助您组织类并构建类与类之间的结构关系。
