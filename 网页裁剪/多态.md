## 多态

*多态（polymorphism）*的字典定义指的是生物学中的一个原理，即一个生物体或物种可以有许多不同的形态或阶段。这一原理同样可以应用于面向对象编程以及像 Java 这样的语言。一个类的子类可以定义自己独有的行为，同时又共享父类的某些相同功能。

可以通过对 `Bicycle` 类做一个小的修改来演示多态。例如，可以为该类添加一个 `printDescription` 方法，用于显示当前存储在实例中的所有数据。

```java
public void printDescription(){
    System.out.println("\nBike is " + "in gear " + this.gear
        + " with a cadence of " + this.cadence +
        " and travelling at a speed of " + this.speed + ". ");
}
```

为了演示 Java 语言中的多态特性，用 `MountainBike` 和 `RoadBike` 类扩展 `Bicycle` 类。对于 `MountainBike`，添加一个 `suspension`（悬挂）字段，它是一个 `String` 值，用于指示自行车是否具有前减震器（`Front`），或者具有前后减震器（`Dual`）。

下面是更新后的类：

```java
public class MountainBike extends Bicycle {
    private String suspension;

    public MountainBike(
               int startCadence,
               int startSpeed,
               int startGear,
               String suspensionType){
        super(startCadence,
              startSpeed,
              startGear);
        this.setSuspension(suspensionType);
    }

    public String getSuspension(){
      return this.suspension;
    }

    public void setSuspension(String suspensionType) {
        this.suspension = suspensionType;
    }

    public void printDescription() {
        super.printDescription();
        System.out.println("The " + "MountainBike has a" +
            getSuspension() + " suspension.");
    }
}
```

注意被重写的 `printDescription` 方法。除了之前提供的信息外，输出中还包含了关于悬挂的额外数据。

接下来，创建 `RoadBike` 类。由于公路车或赛车有细轮胎，因此添加一个用于跟踪轮胎宽度的属性。下面是 `RoadBike` 类：

```java
public class RoadBike extends Bicycle{
    // 单位为毫米（mm）
    private int tireWidth;

    public RoadBike(int startCadence,
                    int startSpeed,
                    int startGear,
                    int newTireWidth){
        super(startCadence,
              startSpeed,
              startGear);
        this.setTireWidth(newTireWidth);
    }

    public int getTireWidth(){
      return this.tireWidth;
    }

    public void setTireWidth(int newTireWidth){
        this.tireWidth = newTireWidth;
    }

    public void printDescription(){
        super.printDescription();
        System.out.println("The RoadBike" + " has " + getTireWidth() +
            " MM tires.");
    }
}
```

注意，`printDescription` 方法再次被重写了。这一次，显示的是有关轮胎宽度的信息。

总结一下，共有三个类：`Bicycle`、`MountainBike` 和 `RoadBike`。两个子类都重写了 `printDescription` 方法，并打印各自独有的信息。

下面是一个测试程序，它创建了三个 `Bicycle` 变量。每个变量被赋值为这三种自行车类之一。然后打印每个变量。

```java
public class TestBikes {
  public static void main(String[] args){
    Bicycle bike01, bike02, bike03;

    bike01 = new Bicycle(20, 10, 1);
    bike02 = new MountainBike(20, 10, 5, "Dual");
    bike03 = new RoadBike(40, 20, 8, 23);

    bike01.printDescription();
    bike02.printDescription();
    bike03.printDescription();
  }
}
```

下面是测试程序的输出：

```text
Bike is in gear 1 with a cadence of 20 and travelling at a speed of 10. 

Bike is in gear 5 with a cadence of 20 and travelling at a speed of 10. 
The MountainBike has a Dual suspension.

Bike is in gear 8 with a cadence of 40 and travelling at a speed of 20. 
The RoadBike has 23 MM tires.
```

Java 虚拟机（JVM）会为每个变量所引用的对象调用相应的方法。它并不会调用由变量类型所定义的方法。这种行为被称为*虚方法调用（virtual method invocation）*，它展示了 Java 语言中重要的多态特性的一个方面。
