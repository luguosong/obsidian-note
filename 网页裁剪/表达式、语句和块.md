## 表达式、语句和块

既然您已经了解了变量和运算符，接下来该学习*表达式(expression)*、*语句(statement)*和*块(block)*了。运算符可用于构建计算值的表达式；表达式是语句的核心组成部分；语句可分组为块。

## 表达式

*表达式(expression)*是由变量、运算符和方法调用按照语言语法构造而成、最终求值为单一值的构造。您已经见过表达式的示例，下面以粗体标出：

```java
int cadence = 0;
anArray[0] = 100;
System.out.println("Element 1 at index 0: " + anArray[0]);

int result = 1 + 2; // result is now 3
if (value1 == value2) 
    System.out.println("value1 == value2");
```

表达式所返回值的数据类型取决于表达式中使用的元素。表达式 `cadence = 0` 返回一个 `int`，因为赋值运算符返回与其左侧操作数相同数据类型的值；在这里，`cadence` 是一个 `int`。正如您从其它表达式中所见，表达式也可以返回其它类型的值，例如 `boolean` 或 `String`。

Java 编程语言允许您由各种较小的表达式构造复合表达式(compound expression)，只要表达式某一部分所需的数据类型与另一部分的数据类型相匹配即可。下面是一个复合表达式的示例：

```text
1 * 2 * 3
```

在这个特定示例中，表达式求值的顺序并不重要，因为乘法的结果与顺序无关；无论您以何种顺序应用这些乘法，结果总是一样的。然而，并非所有表达式都是如此。例如，下面的表达式会给出不同的结果，取决于您是先执行加法还是先执行除法：

```text
x + y / 100    // ambiguous
```

您可以使用配对的括号（`(` 和 `)`）精确指定表达式将如何被求值。例如，要使前一个表达式没有歧义，您可以这样写：

```text
(x + y) / 100  // unambiguous, recommended
```

如果您没有显式地指明操作的执行顺序，那么顺序就由表达式中所使用运算符被赋予的优先级(precedence)决定。优先级较高的运算符先被求值。例如，除法运算符的优先级高于加法运算符。因此，下面两条语句是等价的：

```text
x + y / 100 

x + (y / 100) // unambiguous, recommended
```

在编写复合表达式时，请显式地用括号指明哪些运算符应该先被求值。这种做法使代码更易于阅读和维护。

## 语句

语句大致相当于自然语言中的句子。*语句(statement)*构成一个完整的执行单元。以下类型的表达式可以通过用分号（`;`）结尾而成为语句。

- 赋值表达式
- 对 `++` 或 `--` 的任何使用
- 方法调用
- 对象创建表达式

这样的语句称为*表达式语句(expression statement)*。下面是一些表达式语句的示例。

```java
// assignment statement
aValue = 8933.234;
// increment statement
aValue++;
// method invocation statement
System.out.println("Hello World!");
// object creation statement
Bicycle myBike = new Bicycle();
```

除了表达式语句之外，还有另外两种语句：*声明语句(declaration statement)*和*控制流语句(control flow statement)*。*声明语句*用于声明一个变量。您已经见过许多声明语句的示例：

```java
// declaration statement
double aValue = 8933.234;
```

最后，*控制流语句*调节语句执行的顺序。您将在下一节控制流语句中学到有关控制流语句的内容。

## 块

*块(block)*是位于配对花括号之间的一组零个或多个语句，可以出现在允许使用单个语句的任何地方。下面的示例 [`BlockDemo`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/BlockDemo.java) 说明了块的用法：

```java
class BlockDemo {
     public static void main(String[] args) {
          boolean condition = true;
          if (condition) { // begin block 1
               System.out.println("Condition is true.");
          } // end block one
          else { // begin block 2
               System.out.println("Condition is false.");
          } // end block 2
     }
}
```
