## 学习路径：XML 绑定的 Java 架构

**XML 绑定的 Java 架构（Java Architecture for XML Binding，JAXB）** 学习路径通过 JAXB 应用程序示例介绍 JAXB 技术。

## 阅读本教程之前

要理解并使用 XML 绑定的 Java 架构（JAXB）教程中的信息，您应当具备以下技术的知识：

- Java 编程语言及其开发环境
- 可扩展标记语言（eXtensible Mark-up Language，XML）

本教程仅详细讲解特定于 JAXB 的代码。

[JAXB 简介（Introduction to JAXB）](https://docs.oracle.com/javase/tutorial/jaxb/intro/index.html) 提供对 JAXB 技术的简要描述，包括其用途和主要特性。
