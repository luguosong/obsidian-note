# 国际化

> 文档说明

《Java 教程》(The Java Tutorials) 是基于 JDK 8 编写的。本页所描述的示例与实践未采用后续版本中引入的改进，并且可能使用了目前已不可用的技术。
请参阅 [Dev.java](https://dev.java/learn/)，获取充分利用最新版本的更新版教程。
请参阅 [Java 语言变更](https://docs.oracle.com/pls/topic/lookup?ctx=en/java/javase&id=java_language_changes)，了解 Java SE 9 及后续版本中更新的语言特性摘要。
请参阅 [JDK 发行说明](https://www.oracle.com/technetwork/java/javase/jdk-relnotes-index-2162236.html)，获取所有 JDK 版本的新特性、增强功能以及已移除或弃用的选项的相关信息。

## 路线：国际化

本路线中的课程教你如何对 Java 应用程序进行国际化。国际化后的应用程序可以轻松适应世界各地最终用户的习俗和语言。

---

**注意：** 本教程路线涵盖*核心*国际化功能，这是为桌面、企业和移动应用程序提供的附加特性所必需的基础。有关更多信息，请参阅 [Java 国际化主页](http://www.oracle.com/technetwork/java/javase/tech/intl-139810.html)。

[[国际化-简介|**简介**]] 定义国际化这一术语，给出一个快速示例程序，并提供一个可用于对现有程序进行国际化的检查清单。

[[国际化-区域设置|**设置区域设置**]] 解释如何创建和使用 `Locale` 对象。

[[国际化-格式化|**格式化**]] 解释如何根据 `Locale` 格式化数字、日期和文本消息，以及如何通过模式创建自定义格式。

[[国际化-文本处理|**处理文本**]] 提供以与区域无关的方式操作文本的技术。

[[国际化-服务提供者|**国际化的服务提供者**]] 解释如何启用依赖于区域的数据和服务的插件。
