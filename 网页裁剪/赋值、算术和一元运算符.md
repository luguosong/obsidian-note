## 赋值、算术和一元运算符

## 简单赋值运算符

您会经常遇到的最常见的运算符之一就是简单赋值运算符「`=`」。您在 Bicycle 类中已经见过这个运算符；它将其右侧的值赋给左侧的操作数(operand)：

```java
int cadence = 0;
int speed = 0;
int gear = 1;
```

该运算符也可用于对象，以赋值*对象引用(object reference)*，相关讨论见[创建对象（Creating Objects）](https://docs.oracle.com/javase/tutorial/java/javaOO/objectcreation.html)。

## 算术运算符

Java 编程语言提供了执行加法、减法、乘法和除法的运算符。您很可能通过基础数学中对应的运算符就能认出它们。对您来说唯一可能看起来陌生的符号是「`%`」，它用一个操作数除以另一个操作数，并将余数作为结果返回。

| 运算符 | 描述 |
| --- | --- |
| `+` | 加法运算符（也用于字符串连接） |
| `-` | 减法运算符 |
| `*` | 乘法运算符 |
| `/` | 除法运算符 |
| `%` | 取余运算符 |

下面的程序 [`ArithmeticDemo`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/ArithmeticDemo.java) 测试了算术运算符。

```java
class ArithmeticDemo {

    public static void main (String[] args) {

        int result = 1 + 2;
        // result 现在是 3
        System.out.println("1 + 2 = " + result);
        int original_result = result;

        result = result - 1;
        // result 现在是 2
        System.out.println(original_result + " - 1 = " + result);
        original_result = result;

        result = result * 2;
        // result 现在是 4
        System.out.println(original_result + " * 2 = " + result);
        original_result = result;

        result = result / 2;
        // result 现在是 2
        System.out.println(original_result + " / 2 = " + result);
        original_result = result;

        result = result + 8;
        // result 现在是 10
        System.out.println(original_result + " + 8 = " + result);
        original_result = result;

        result = result % 7;
        // result 现在是 3
        System.out.println(original_result + " % 7 = " + result);
    }
}
```

该程序输出如下：

```text
1 + 2 = 3
3 - 1 = 2
2 * 2 = 4
4 / 2 = 2
2 + 8 = 10
10 % 7 = 3
```

您还可以将算术运算符与简单赋值运算符结合使用，以创建*复合赋值(compound assignment)*。例如，`x+=1;` 和 `x=x+1;` 都会将 `x` 的值增加 1。

`+` 运算符还可用于将两个字符串连接（拼接）在一起，如下面的 [`ConcatDemo`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/ConcatDemo.java) 程序所示：

```java
class ConcatDemo {
    public static void main(String[] args){
        String firstString = "This is";
        String secondString = " a concatenated string.";
        String thirdString = firstString+secondString;
        System.out.println(thirdString);
    }
}
```

在该程序结束时，变量 `thirdString` 包含 "This is a concatenated string."，该字符串会被打印到标准输出。

## 一元运算符

一元运算符只需要一个操作数；它们执行各种操作，例如将一个值递增/递减 1、对表达式取反，或反转布尔值。

| 运算符 | 描述 |
| --- | --- |
| `+` | 一元加运算符；表示正值（不过数字本身就是正的） |
| `-` | 一元减运算符；对表达式取反 |
| `++` | 递增运算符；将值加 1 |
| `--` | 递减运算符；将值减 1 |
| `!` | 逻辑补运算符；反转布尔值 |

下面的程序 [`UnaryDemo`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/UnaryDemo.java) 测试了一元运算符：

```java
class UnaryDemo {

    public static void main(String[] args) {

        int result = +1;
        // result 现在是 1
        System.out.println(result);

        result--;
        // result 现在是 0
        System.out.println(result);

        result++;
        // result 现在是 1
        System.out.println(result);

        result = -result;
        // result 现在是 -1
        System.out.println(result);

        boolean success = false;
        // false
        System.out.println(success);
        // true
        System.out.println(!success);
    }
}
```

递增/递减运算符可以应用于操作数之前（前缀）或之后（后缀）。代码 `result++;` 和 `++result;` 最终都会使 `result` 递增 1。唯一的区别在于，前缀版本（`++result`）求值为递增后的值，而后缀版本（`result++`）求值为原始值。如果您只是执行简单的递增/递减，选择哪个版本其实无关紧要。但如果您将该运算符用在更大的表达式中，您所选择的版本可能会产生显著差异。

下面的程序 [`PrePostDemo`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/PrePostDemo.java) 演示了前缀/后缀一元递增运算符：

```java
class PrePostDemo {
    public static void main(String[] args){
        int i = 3;
        i++;
        // 打印 4
        System.out.println(i);
        ++i;               
        // 打印 5
        System.out.println(i);
        // 打印 6
        System.out.println(++i);
        // 打印 6
        System.out.println(i++);
        // 打印 7
        System.out.println(i);
    }
}
```
