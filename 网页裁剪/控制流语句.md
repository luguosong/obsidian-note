## 控制流语句

源文件中的语句通常按照它们出现的顺序，从上到下执行。然而，*控制流语句(control flow statement)*通过采用决策(decision making)、循环(looping)和分支(branching)来打断执行流程，使您的程序能够*有条件地*执行特定的代码块。本节描述 Java 编程语言所支持的决策语句（`if-then`、`if-then-else`、`switch`）、循环语句（`for`、`while`、`do-while`）以及分支语句（`break`、`continue`、`return`）。
