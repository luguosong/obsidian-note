## 嵌套类问题与练习

## 问题

1. 程序 [`Problem.java`](https://docs.oracle.com/javase/tutorial/java/javaOO/QandE/Problem.java) 无法编译。您需要做什么才能让它编译？为什么？
2. 使用 [`Box`](https://docs.oracle.com/javase/8/docs/api/javax/swing/Box.html) 类（位于 `javax.swing` 包中）的 Java API 文档来帮助您回答以下问题。
	1. `Box` 定义了哪个静态嵌套类？
		2. `Box` 定义了哪个内部类？
		3. `Box` 的内部类的超类是什么？
		4. `Box` 的哪些嵌套类可以从任何类中使用？
		5. 如何创建 `Box` 的 `Filler` 类的实例？

## 练习

1. 获取文件 [`Class1.java`](https://docs.oracle.com/javase/tutorial/java/javaOO/QandE/Class1.java)。编译并运行 `Class1`。输出是什么？
2. 以下练习涉及修改类 [`DataStructure.java`](https://docs.oracle.com/javase/tutorial/java/javaOO/examples/DataStructure.java)，即[内部类示例（Inner Class Example）](https://docs.oracle.com/javase/tutorial/java/javaOO/innerclasses.html)一节中讨论的那个类。
	1. 定义一个名为 `print(DataStructureIterator iterator)` 的方法。用 `EvenIterator` 类的实例调用此方法，使其执行与方法 `printEven` 相同的功能。
		2. 调用方法 `print(DataStructureIterator iterator)`，使其打印具有奇数索引值的元素。使用匿名类作为该方法的参数，而不是接口 `DataStructureIterator` 的实例。
		3. 定义一个名为 `print(java.util.function.Function<Integer, Boolean> iterator)` 的方法，执行与 `print(DataStructureIterator iterator)` 相同的功能。用 Lambda 表达式调用此方法，打印具有偶数索引值的元素。再用另一个 Lambda 表达式调用此方法，打印具有奇数索引值的元素。
		4. 定义两个方法，使下面两条语句分别打印具有偶数索引值的元素和具有奇数索引值的元素：
		```text
		DataStructure ds = new DataStructure()
		// ...
		ds.print(DataStructure::isEvenIndex);
		ds.print(DataStructure::isOddIndex);
		```
