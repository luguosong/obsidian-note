## 声明类

您已经见过以如下方式定义的类：

```text
class MyClass {
    // field, constructor, and 
    // method declarations
}
```

这是一个*类声明(class declaration)*。*类体(class body)*（花括号之间的区域）包含了为从该类创建的对象提供生命周期的所有代码：用于初始化新对象的构造方法、提供该类及其对象状态的字段声明，以及实现该类及其对象行为的方法。

前面的类声明是一个最小化的声明。它只包含了类声明所必需的那些组成部分。您可以在类声明的开头提供关于该类的更多信息，例如其父类的名称、它是否实现了任何接口等等。例如，

```text
class MyClass extends MySuperClass implements YourInterface {
    // field, constructor, and
    // method declarations
}
```

表示 `MyClass` 是 `MySuperClass` 的子类，并且它实现了 `YourInterface` 接口。

您还可以在最开头添加诸如 *public* 或 *private* 这样的修饰符(modifier)——因此您可以看到，类声明的开头一行可能会变得相当复杂。决定哪些其它类可以访问 `MyClass` 的 *public* 和 *private* 修饰符将在本课稍后讨论。关于接口和继承的课程将解释在类声明中如何以及为什么要使用 *extends* 和 *implements* 关键字。目前您不必关心这些额外的复杂性。

一般而言，类声明可以按顺序包含以下组成部分：

1. 修饰符，如 *public*、*private*，以及稍后您将遇到的其它若干修饰符。（不过请注意，*private* 修饰符只能应用于[嵌套类（Nested Classes）](https://docs.oracle.com/javase/tutorial/java/javaOO/nested.html)。）
2. 类名，按约定首字母大写。
3. 该类的父类（超类）的名称（如果有的话），前面加上关键字 *extends*。一个类只能*扩展(extend)*（即作为子类继承）一个父类。
4. 该类所实现的接口的逗号分隔列表（如果有的话），前面加上关键字 *implements*。一个类可以*实现(implement)*多个接口。
5. 类体，由花括号 {} 包围。
