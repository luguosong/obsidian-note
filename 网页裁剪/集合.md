## 路线：集合

本节介绍 Java 集合框架(Java Collections Framework)。在这里，你将了解什么是集合，以及它们如何让你的工作更轻松、程序更优秀。你将学习构成 Java 集合框架的核心元素——接口(interfaces)、实现(implementations)、聚合操作(aggregate operations)和算法(algorithms)。

[[集合-简介|**简介**]] 告诉你什么是集合，以及它们如何让你的工作更轻松、程序更优秀。你将了解构成集合框架的核心元素：*接口*、*实现*和*算法*。

[[集合-接口|**接口**]] 描述*核心集合接口*，这是 Java 集合框架的核心与灵魂。你将学习有效使用这些接口的一般准则，包括何时使用哪个接口。你还将学习每个接口的惯用法，帮助你充分发挥接口的作用。

[[聚合操作|**聚合操作**]] 代表你遍历集合，使你能编写更简洁高效的代码来处理存储在集合中的元素。

[[集合-实现|**实现**]] 描述 JDK 的*通用集合实现*，并告诉你何时使用哪种实现。你还将学习*包装器实现*，它们为通用实现添加功能。

[[集合-算法|**算法**]] 描述 JDK 提供的用于操作集合的*多态算法*。运气好的话，你再也不必自己编写排序例程了！

[[集合-自定义实现|**自定义实现**]] 告诉你为什么可能想编写自己的集合实现（而不是使用 JDK 提供的通用实现），以及如何去做。借助 JDK 的*抽象集合实现*，这很容易！

[[集合-互操作性|**互操作性**]] 告诉你集合框架如何与在集合被加入 Java 之前就存在的旧 API 互操作。此外，它还告诉你如何设计新 API，使其能与其他新 API 无缝互操作。
