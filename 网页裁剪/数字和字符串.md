## 课程：数字和字符串

## 数字

本节首先讨论（`java.lang` 包中的）`Number` 类及其子类。具体而言，本节讲述在何种情况下应使用这些类的实例化而非原始数据类型。此外，本节还讨论您在处理数字时可能需要用到的其他类，例如格式化或使用数学函数以补充语言内置的运算符。最后，讨论自动装箱(autoboxing)与拆箱(unboxing)，这是一项可简化代码的编译器特性。

## 字符串

字符串在 Java 编程中被广泛使用，它是一系列字符。在 Java 编程语言中，字符串是对象。本节描述如何使用 `String` 类来创建和操作字符串。它还对比了 `String` 和 `StringBuilder` 类。
