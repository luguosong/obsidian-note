## 课程：深入部署

富互联网应用程序(RIA) 部署涉及各种技术和工具，如 Java 网络启动协议(JNLP)、部署工具包、pack200、jarsigner 等。本课探索这些技术和工具如何帮助开发者部署 RIA（applet 和 Java Web Start 应用程序）。

涉及 RIA 部署的主要组件如下：

- 将部署 RIA 的 HTML 页面
- RIA 的 JNLP 文件
- 包含 RIA 类文件和资源的 JAR 文件
