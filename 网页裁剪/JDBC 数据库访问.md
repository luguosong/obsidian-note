## 学习路径：JDBC 数据库访问

JDBC API 的设计宗旨是让简单的事情保持简单。这意味着 JDBC 使日常的数据库任务变得容易。本学习路径带您通过示例学习如何使用 JDBC 执行常见的 SQL 语句，并完成数据库应用程序中常见的其它目标。

本学习路径分为以下课程：

[**JDBC 简介（JDBC Introduction）**](https://docs.oracle.com/javase/tutorial/jdbc/overview/index.html) 列出 JDBC 的特性，描述 JDBC 架构，并回顾 SQL 命令和关系数据库的概念。

[**JDBC 基础（JDBC Basics）**](https://docs.oracle.com/javase/tutorial/jdbc/basics/index.html) 涵盖 JDBC API。

在第一节课结束时，您将知道如何使用基本的 JDBC API 创建表、向表中插入值、查询表、检索查询结果并更新表。在此过程中，您将学习如何使用简单语句和预编译语句（prepared statements），并看到一个存储过程的示例。您还将学习如何执行事务以及如何捕获异常和警告。
