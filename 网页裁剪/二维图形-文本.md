## 课程：使用文本 API

本课向你介绍使用文本 API 应用文本渲染能力的概念。在本路线中，你已经使用了基本的 Java 2D 文本 API，并知道如何设置字体和位置以及如何绘制文本。

本课在此基础上扩展，帮助你理解如何使用这些 API，并进一步深入了解 Java 2D 文本显示的能力。

以下各节讨论这些主题。

[[二维图形-物理与逻辑字体|物理与逻辑字体]]

本节解释如何使用 `Font` 类的方法确定系统上可用的字体、创建 `Font` 对象以及获取有关字体族的信息。

[[二维图形-测量文本|测量文本]]

本节解释如何使用 `FontMetrics` 类的实例正确测量文本。

[[二维图形-高级文本显示|高级文本显示]]

本节解释如何定位和渲染一段带样式的文本、显示抗锯齿文本、使用文本属性设置文本样式以及处理双向文本。
