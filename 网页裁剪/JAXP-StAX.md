## 课程：XML 流式 API

本课关注 XML 流式 API(StAX)，一种基于流式 Java 技术、事件驱动的拉取解析 API，用于读写 XML 文档。StAX 使你能创建快速、相对易于编程且内存占用小的双向 XML 解析器。
