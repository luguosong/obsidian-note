## 对象问题与练习

## 问题

1. 下面的程序有什么问题？
	```java
	public class SomethingIsWrong {
	    public static void main(String[] args) {
	        Rectangle myRect;
	        myRect.width = 40;
	        myRect.height = 50;
	        System.out.println("myRect's area is " + myRect.area());
	    }
	}
	```
2. 下面的代码创建了一个数组和一个字符串对象。在这段代码执行之后，存在多少个指向这些对象的引用？其中是否有对象符合垃圾回收的条件？
	```java
	...
	String[] students = new String[10];
	String studentName = "Peter Parker";
	students[0] = studentName;
	studentName = null;
	...
	```
3. 程序如何销毁它所创建的对象？

## 练习

1. 修复问题 1 中名为 `SomethingIsWrong` 的程序。
2. 给定下面的类，名为 [`NumberHolder`](https://docs.oracle.com/javase/tutorial/java/javaOO/QandE/NumberHolder.java)，编写一些代码来创建该类的实例，初始化其两个成员变量，然后显示每个成员变量的值。
	```java
	public class NumberHolder {
	    public int anInt;
	    public float aFloat;
	}
	```
