## Java 技术将如何改变我的生活？ (How Will Java Technology Change My Life?)

我们无法承诺，学习 Java 编程语言就能让你名利双收，甚至找到一份工作。不过，它很可能让你的程序变得更好，而且相比其他语言只需更少的精力。我们相信 Java 技术将帮助你做到以下几点：

- **快速上手 (Get started quickly)**：尽管 Java 编程语言是一门强大的面向对象语言，但它很容易学习，特别是对于已经熟悉 C 或 C++ 的程序员而言。
- **编写更少的代码 (Write less code)**：对程序度量指标（类的数量、方法的数量等）的比较表明，用 Java 编程语言编写的程序可以比用 C++ 编写的同一程序小四倍。
- **编写更好的代码 (Write better code)**：Java 编程语言鼓励良好的编码实践，自动垃圾回收(garbage collection)机制帮助你避免内存泄漏。它的面向对象特性、JavaBeans™ 组件架构以及广泛且易于扩展的 API，使你能够重用现有的、经过测试的代码，从而引入更少的缺陷。
- **更快速地开发程序 (Develop programs more quickly)**：Java 编程语言比 C++ 更简单，因此用 Java 编写时的开发时间最多可以快两倍。你的程序也将需要更少的代码行。
- **避免平台依赖 (Avoid platform dependencies)**：通过避免使用用其他语言编写的库，你可以保持程序的可移植性。
- **一次编写，到处运行 (Write once, run anywhere)**：由于用 Java 编程语言编写的应用程序被编译成与机器无关的字节码(bytecodes)，因此它们可以在任何 Java 平台上一致地运行。
- **更轻松地分发软件 (Distribute software more easily)**：借助 Java Web Start 软件，用户只需单击鼠标即可启动你的应用程序。启动时的自动版本检查可确保用户始终使用你软件的最新版本。如果有更新可用，Java Web Start 软件将自动更新他们的安装版本。
