## Java 技术能做什么？ (What Can Java Technology Do?)

通用的、高级的 Java 编程语言是一个强大的软件平台。Java 平台的每一个完整实现都为你提供以下功能：

- **开发工具 (Development Tools)**：开发工具提供了编译、运行、监控、调试和编写文档所需的一切。作为一名新开发者，你将使用的主要工具是 `javac` 编译器、`java` 启动器和 `javadoc` 文档工具。
- **应用程序编程接口 (Application Programming Interface, API)**：API 提供 Java 编程语言的核心功能。它提供了大量现成的、可供你自己的应用程序直接使用的有用类。其涵盖范围从基本对象，到网络和安全，再到 XML 生成和数据库访问等等。核心 API 非常庞大；要了解其包含内容的概览，请查阅 [Java Platform Standard Edition 8 文档](https://docs.oracle.com/javase/8/docs/index.html)。
- **部署技术 (Deployment Technologies)**：JDK 软件提供了标准机制，例如 Java Web Start 软件和 Java 插件(Java Plug-In)软件，用于将你的应用程序部署给最终用户。
- **用户界面工具包 (User Interface Toolkits)**：JavaFX、Swing 和 Java 2D 工具包使得创建复杂的图形用户界面(Graphical User Interfaces, GUI)成为可能。
- **集成库 (Integration Libraries)**：集成库（如 Java IDL API、JDBC API、Java 命名与目录接口(Java Naming and Directory Interface, JNDI) API、Java RMI，以及基于 Internet Inter-ORB 协议的 Java 远程方法调用技术(Java RMI-IIOP Technology)）支持数据库访问和远程对象的操作。
