## 问题与练习：面向对象编程概念

## 问题

1. 真实世界的对象包含 \_\_\_ 和 \_\_\_。
2. 软件对象存储其状态在 \_\_\_ 中。
3. 软件对象通过 \_\_\_ 来展现其行为。
4. 将内部数据对外部世界隐藏，仅通过公开暴露的方法访问它，这种方式被称为数据 \_\_\_。
5. 软件对象的图纸被称为 \_\_\_。
6. 公共行为可以在一个 \_\_\_ 中定义，并使用 \_\_\_ 关键字继承到一个 \_\_\_ 中。
7. 一组没有实现的方法集合被称为 \_\_\_。
8. 按功能组织类和接口的命名空间被称为 \_\_\_。
9. 术语 API 代表 \_\_\_？

## 练习

1. 为您在本学习路径开头观察到的每个真实世界对象创建新的类。如果忘记了所需语法，请参考 Bicycle 类。
2. 对于您在上面创建的每个新类，创建一个定义其行为的接口，然后要求您的类实现它。省略一两个方法并尝试编译。错误是什么样的？
