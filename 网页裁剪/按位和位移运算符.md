## 按位和位移运算符

Java 编程语言还提供了对整型(integral type)执行按位和位移操作的运算符。本节讨论的运算符较少使用。因此，对其的介绍较为简略；目的仅在于让您知道这些运算符的存在。

一元按位取补运算符「`~`」反转位模式(bit pattern)；它可以应用于任何整型，将每个「0」变为「1」，将每个「1」变为「0」。例如，一个 `byte` 包含 8 位；将该运算符应用于位模式为 "00000000" 的值，会将其模式变为 "11111111"。

有符号左移运算符「`<<`」将位模式向左移位，有符号右移运算符「`>>`」将位模式向右移位。位模式由左侧操作数给出，移位的位数由右侧操作数给出。无符号右移运算符「`>>>`」将 0 移入最左边的位置，而 `">>"` 之后最左边的位置取决于符号扩展(sign extension)。

按位 `&` 运算符执行按位与操作。

按位 `^` 运算符执行按位异或操作。

按位 `|` 运算符执行按位或操作。

下面的程序 [`BitDemo`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/BitDemo.java) 使用按位与运算符向标准输出打印数字 "2"。

```java
class BitDemo {
    public static void main(String[] args) {
        int bitmask = 0x000F;
        int val = 0x2222;
        // 打印 "2"
        System.out.println(val & bitmask);
    }
}
```
