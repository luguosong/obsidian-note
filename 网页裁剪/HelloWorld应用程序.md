## 课程："Hello World!"应用程序 (Lesson: The "Hello World!" Application)

以下各节提供了编译并运行一个简单"Hello World!"应用程序的详细说明。第一节介绍如何上手 NetBeans IDE——一个能极大简化软件开发过程的集成开发环境(integrated development environment, IDE)。NetBeans IDE 可以在下面列出的所有平台上运行。其余各节提供了在无集成开发环境的情况下入门的、特定于平台的操作说明。如果你遇到问题，务必查阅常见问题(common problems)一节；它为新用户遇到的许多问题提供了解决方案。

[[NetBeans版HelloWorld]]这些说明面向 NetBeans IDE 的用户。NetBeans IDE 运行在 Java 平台之上，这意味着你可以在任何提供了 JDK 7 的操作系统上使用它。这些操作系统包括 Microsoft Windows、Solaris OS、Linux 和 Mac OS X。只要可能，我们推荐使用 NetBeans IDE 而非命令行。

[[Windows版HelloWorld]] 这些命令行说明面向 Windows XP Professional、Windows XP Home、Windows Server 2003、Windows 2000 Professional 和 Windows Vista 的用户。

[[Solaris-Linux-Mac版HelloWorld]] 这些命令行说明面向 Solaris OS、Linux 和 Mac OS X 的用户。
