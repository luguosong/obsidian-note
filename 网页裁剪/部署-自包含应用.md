## 课程：部署自包含应用程序

自包含应用程序由一个单独的、可安装的包组成，其中包含你的应用程序和运行该应用程序所需的 JRE 副本。安装该应用程序时，它的行为与任何本机应用程序相同。为用户提供自包含应用程序可以避免在浏览器中运行应用程序相关的安全问题。

你可以通过提供自己的图标来自定义自包含应用程序。可以设置文件关联，以便当用户打开你的应用程序可以处理的文件时，你的应用程序会自动启动。支持多个入口点，因此你可以在单个自包含应用程序包中交付一套应用程序。

可以使用 Java 打包工具打包自包含应用程序。`javapackager` 命令从命令行创建自包含应用程序的包。NetBeans 也可用于创建自包含应用程序包。本课描述如何使用 Ant 任务创建包。
