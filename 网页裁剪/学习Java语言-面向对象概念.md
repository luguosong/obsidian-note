## 课程：面向对象编程概念

如果你以前从未使用过面向对象编程语言，那么在开始编写任何代码之前，你需要先学习一些基本概念。本课将向你介绍对象、类、继承、接口和包。每个讨论都关注这些概念如何与现实世界相关联，同时介绍 Java 编程语言的语法。

## 什么是对象？

对象是相关状态和行为的软件包。软件对象常用于模拟你在日常生活中发现的现实世界对象。本节解释状态和行为如何在对象内表示，介绍数据封装的概念，并解释以这种方式设计软件的好处。

## 什么是类？

类是从中创建对象的蓝图或原型。本节定义一个模拟现实世界对象状态和行为的类。它有意关注基础知识，展示即使是简单的类也能清晰地模拟状态和行为。

## 什么是继承？

继承为组织和结构化你的软件提供了一种强大而自然的机制。本节解释类如何从其超类继承状态和行为，并解释如何使用 Java 编程语言提供的简单语法从一个类派生另一个类。

## 什么是接口？

接口是类与外部世界之间的契约。当一个类实现一个接口时，它承诺提供该接口公开的行为。本节定义一个简单的接口，并解释实现它的任何类所需的必要更改。

## 什么是包？

包是一种以逻辑方式组织类和接口的命名空间。将你的代码放入包中使大型软件项目更易于管理。本节解释为什么这很有用，并向你介绍 Java 平台提供的应用程序编程接口(API)。
