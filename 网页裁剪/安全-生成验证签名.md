## 课程：生成和验证签名

本课逐步带你完成使用 JDK 安全 API 为数据生成数字签名并验证签名是否真实所需的步骤。本课面向希望将安全功能（包括加密服务）纳入其程序的开发者。

本课演示 JDK 安全 API 在签名文档方面的使用。本课展示由拥有原始文档的人执行的一个程序会做什么来生成密钥、使用私钥为文档生成数字签名，并将公钥和签名导出到文件。

然后它展示由文档、签名和公钥的接收者执行的另一个程序的示例。它展示程序如何导入公钥并验证签名的真实性。本课还讨论并演示供应和导入密钥（包括在证书中）的可行替代方案和方法。
