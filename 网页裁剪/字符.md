## 字符

大多数时候，如果你使用的是单个字符值，你会使用基本类型 `char`。例如：

```java
char ch = 'a'; 
// 大写希腊字母 omega 字符的 Unicode
char uniChar = '\u03A9';
// 一个 char 数组
char[] charArray = { 'a', 'b', 'c', 'd', 'e' };
```

然而，有时你需要将 char 作为对象使用——例如，作为期望对象的方法参数。Java 编程语言为此提供了一个*包装类（wrapper class）*，将 `char`"包装"在 `Character` 对象中。一个 `Character` 类型的对象包含单个字段，其类型为 `char`。这个 [Character](https://docs.oracle.com/javase/8/docs/api/java/lang/Character.html) 类还提供了许多有用的类（即静态）方法用于操作字符。

你可以使用 `Character` 构造方法创建一个 `Character` 对象：

```java
Character ch = new Character('a');
```

在某些情况下，Java 编译器也会为你创建 `Character` 对象。例如，如果你将一个基本类型 `char` 传递给一个期望对象的方法，编译器会自动为你将 `char` 转换为 `Character`。这一特性被称为*自动装箱（autoboxing）*——如果转换方向相反，则称为*拆箱（unboxing）*。有关自动装箱和拆箱的更多信息，请参见[自动装箱和拆箱（Autoboxing and Unboxing）](https://docs.oracle.com/javase/tutorial/java/data/autoboxing.html)。

---

**注意：** `Character` 类是不可变的，因此一旦创建，`Character` 对象就不能被更改。

---

下表列出了 `Character` 类中一些最有用的方法，但并不详尽。有关此类所有方法的完整列表（超过 50 个），请参阅 [java.lang.Character](https://docs.oracle.com/javase/8/docs/api/java/lang/Character.html) API 规范。

| 方法 | 描述 |
| --- | --- |
| `boolean isLetter(char ch)   boolean isDigit(char ch)` | 分别确定指定的 char 值是字母还是数字。 |
| `boolean isWhitespace(char ch)` | 确定指定的 char 值是否为空白。 |
| `boolean isUpperCase(char ch)   boolean isLowerCase(char ch)` | 分别确定指定的 char 值是大写还是小写。 |
| `char toUpperCase(char ch)   char toLowerCase(char ch)` | 返回指定 char 值的大写或小写形式。 |
| `toString(char ch)` | 返回表示指定字符值的 `String` 对象——即一个单字符的字符串。 |

## 转义序列

前面带有反斜杠（\\）的字符是一个*转义序列（escape sequence）*，对编译器有特殊含义。下表显示了 Java 的转义序列：

| 转义序列 | 描述 |
| --- | --- |
| `\t` | 在此处插入一个制表符。 |
| `\b` | 在此处插入一个退格符。 |
| `\n` | 在此处插入一个换行符。 |
| `\r` | 在此处插入一个回车符。 |
| `\f` | 在此处插入一个换页符。 |
| `\'` | 在此处插入一个单引号字符。 |
| `\"` | 在此处插入一个双引号字符。 |
| `\\` | 在此处插入一个反斜杠字符。 |

当在 print 语句中遇到转义序列时，编译器会相应地解释它。例如，如果你想在引号内再放引号，你必须在内层引号上使用转义序列 \\"。要打印这个句子

```text
She said "Hello!" to me.
```

你应编写

```java
System.out.println("She said \"Hello!\" to me.");
```
