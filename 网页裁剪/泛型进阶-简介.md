## 简介

JDK 5.0 引入了对 Java 编程语言的几项新扩展。其中之一是引入了*泛型(generics)*。

本路线是对泛型的介绍。你可能熟悉其他语言中的类似构造，最著名的是 C++ 模板。如果是这样，你会看到既有相似之处也有重要的区别。如果你不熟悉其他地方的外观类似的构造，那就更好了；你可以从头开始，不必纠正任何误解。
