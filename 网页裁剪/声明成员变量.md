## 声明成员变量

变量有以下几种：

- 类中的成员变量——这些被称为*字段(field)*。
- 方法或代码块中的变量——这些被称为*局部变量(local variable)*。
- 方法声明中的变量——这些被称为*参数(parameter)*。

`Bicycle` 类使用下面几行代码来定义它的字段：

```java
public int cadence;
public int gear;
public int speed;
```

字段声明由三个组成部分构成，顺序如下：

1. 零个或多个修饰符，例如 `public` 或 `private`。
2. 字段的类型。
3. 字段的名称。

`Bicycle` 的字段分别命名为 `cadence`、`gear` 和 `speed`，且数据类型都是整数（`int`）。`public` 关键字将这些字段标识为公有成员，任何能访问该类的对象都可以访问它们。

## 访问修饰符

使用的第一个（最左边的）修饰符让您能够控制哪些其它类可以访问某个成员字段。目前只考虑 `public` 和 `private`。其它访问修饰符将在稍后讨论。

- `public` 修饰符——该字段可以被所有类访问。
- `private` 修饰符——该字段只能在其自身所在的类中被访问。

本着封装(encapsulation)的精神，通常会将字段声明为 private。这意味着它们只能从 Bicycle 类中*直接*访问。然而，我们仍然需要访问这些值。这可以通过添加公有的方法来*间接*实现，这些方法为我们获取字段的值：

```java
public class Bicycle {
        
    private int cadence;
    private int gear;
    private int speed;
        
    public Bicycle(int startCadence, int startSpeed, int startGear) {
        gear = startGear;
        cadence = startCadence;
        speed = startSpeed;
    }
        
    public int getCadence() {
        return cadence;
    }
        
    public void setCadence(int newValue) {
        cadence = newValue;
    }
        
    public int getGear() {
        return gear;
    }
        
    public void setGear(int newValue) {
        gear = newValue;
    }
        
    public int getSpeed() {
        return speed;
    }
        
    public void applyBrake(int decrement) {
        speed -= decrement;
    }
        
    public void speedUp(int increment) {
        speed += increment;
    }
}
```

## 类型

所有变量都必须有一个类型。您可以使用基本类型，如 `int`、`float`、`boolean` 等；也可以使用引用类型，如字符串、数组或对象。

## 变量名

所有变量，无论它们是字段、局部变量还是参数，都遵循语言基础课程 [变量——命名（Variables—Naming）](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/variables.html#naming)中介绍过的相同命名规则和约定。

在本课中，请注意方法名和类名使用相同的命名规则和约定，不同的是：

- 类名的首字母应当大写；
- 方法名中的第一个（或唯一一个）单词应当是一个动词。
