## 课程：语言基础

## 变量

[[变量|变量]] —— 您已经了解到对象将状态存储在字段中。然而，Java 编程语言也使用「变量(variable)」这一术语。本节讨论这一关系，以及变量的命名规则和约定、基本数据类型（原始类型(primitive type)、字符串和数组）、默认值(default value)和字面量(literal)。

## 运算符

[[运算符|运算符]] —— 本节描述 Java 编程语言的运算符。它先介绍最常用的运算符，最后介绍不太常用的运算符。每部分讨论都包含可编译和运行的代码示例。

## 表达式、语句和块

[[表达式、语句和块|表达式、语句和块]] —— 运算符可用于构建计算值的表达式(expression)；表达式是语句(statement)的核心组成部分；语句可分组为块(block)。本节使用您已经见过的示例代码讨论表达式、语句和块。

## 控制流语句

[[控制流语句|控制流语句]] —— 本节描述 Java 编程语言支持的控制流语句(control flow statement)。涵盖使您的程序能够有条件地执行特定代码块的决策(decision-making)、循环(looping)和分支(branching)语句。
