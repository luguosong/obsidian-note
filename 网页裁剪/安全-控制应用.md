## Lesson: Quick Tour of Controlling Applications

---

**Pre-requisite lesson:** [Creating a Policy File](https://docs.oracle.com/javase/tutorial/security/tour1/index.html)

---

This lesson shows how to use a security manager to grant or deny access to system resources for applications. The lesson also shows how resource accesses, such as reading or writing a file, are not permitted for applications that are run with a security manager unless explicitly allowed by a permission entry in a policy file.

The steps for this lesson are:

1. [Observe Application Freedom](https://docs.oracle.com/javase/tutorial/security/tour2/step1.html)
2. [See How to Restrict Applications](https://docs.oracle.com/javase/tutorial/security/tour2/step2.html)
3. [Set Up the Policy File to Grant the Required Permissions](https://docs.oracle.com/javase/tutorial/security/tour2/step3.html)
4. [See the Policy File Effects](https://docs.oracle.com/javase/tutorial/security/tour2/step4.html)
