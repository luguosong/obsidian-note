## 课程：并发

计算机用户理所当然地认为他们的系统能够同时做多件事。他们假设自己可以继续在字处理程序中工作，而其他应用程序下载文件、管理打印队列并流式传输音频。即使是单个应用程序也经常被期望同时做多件事。例如，那个流式音频应用程序必须同时从网络读取数字音频、解压缩、管理播放并更新其显示。即使是字处理程序也应该始终准备好响应键盘和鼠标事件，无论它有多忙于重新格式化文本或更新显示。能够做到这些事情的软件被称为*并发(concurrent)*软件。

Java 平台从底层就被设计为支持并发编程，Java 编程语言和 Java 类库都提供了基本的并发支持。自 5.0 版本起，Java 平台还包含了高级并发 API。本课介绍平台的基本并发支持，并概述 `java.util.concurrent` 包中的部分高级 API。
