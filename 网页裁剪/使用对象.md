## 使用对象

一旦创建了对象，您大概会想用它来做点什么。您可能需要使用其某个字段的值、更改其某个字段，或者调用其某个方法来执行某个操作。

## 引用对象的字段

对象的字段通过其名称来访问。您必须使用一个明确的、无歧义的名称。

在字段自身所在的类中，您可以为字段使用简单名称(simple name)。例如，我们可以在 `Rectangle` 类*内部*添加一条语句，打印 `width` 和 `height`：

```java
System.out.println("Width and height are: " + width + ", " + height);
```

在这种情况下，`width` 和 `height` 就是简单名称。

对象类之外的代码必须使用一个对象引用或表达式，后跟点（`.`）运算符，再跟一个简单的字段名，例如：

```text
objectReference.fieldName
```

例如，CreateObjectDemo 类中的代码位于 Rectangle 类的代码之外。因此，要引用名为 rectOne 的 Rectangle 对象中的 origin、width 和 height 字段，CreateObjectDemo 类必须分别使用名称 rectOne.origin、rectOne.width 和 rectOne.height。该程序使用其中两个名称来显示 rectOne 的宽度和高度：

```java
System.out.println("Width of rectOne: "  + rectOne.width);
System.out.println("Height of rectOne: " + rectOne.height);
```

试图从 CreateObjectDemo 类的代码中使用简单名称 width 和 height 是没有意义的——这些字段只存在于对象内部——并会导致编译器错误。

随后，程序使用类似的代码显示关于 rectTwo 的信息。相同类型的对象各自拥有相同实例字段的一份副本。因此，每个 Rectangle 对象都有名为 origin、width 和 height 的字段。当您通过对象引用访问一个实例字段时，您引用的就是那个特定对象的字段。CreateObjectDemo 程序中的 rectOne 和 rectTwo 这两个对象拥有不同的 origin、width 和 height 字段。

要访问字段，您可以使用指向对象的命名引用（如前面的示例所示），也可以使用任何返回对象引用的表达式。回想一下，`new` 运算符会返回一个对象引用。因此您可以使用 `new` 返回的值来访问新对象的字段：

```java
int height = new Rectangle().height;
```

这条语句创建一个新的 Rectangle 对象，并立即获取它的高度。本质上，该语句计算了 Rectangle 的默认高度。请注意，此语句执行之后，程序不再拥有对所创建 Rectangle 的引用，因为程序从未在任何地方存储该引用。该对象成为无引用对象，其资源可由 Java 虚拟机自由回收。

## 调用对象的方法

您同样使用对象引用来调用对象的方法。您将方法的简单名称附加到对象引用之后，中间用一个点运算符（`.`）连接。此外，您在包围的圆括号内提供该方法的任何参数。如果该方法不需要任何参数，则使用空的圆括号。

```text
objectReference.methodName(argumentList);
```

或者：

```text
objectReference.methodName();
```

Rectangle 类有两个方法：用于计算矩形面积的 `getArea()` 和用于更改矩形原点的 `move()`。下面是调用这两个方法的 CreateObjectDemo 代码：

```java
System.out.println("Area of rectOne: " + rectOne.getArea());
...
rectTwo.move(40, 72);
```

第一条语句调用 rectOne 的 `getArea()` 方法并显示结果。第二行移动 rectTwo，因为 `move()` 方法为该对象的 origin.x 和 origin.y 赋了新值。

与实例字段一样，*objectReference* 必须是一个指向对象的引用。您可以使用一个变量名，也可以使用任何返回对象引用的表达式。`new` 运算符会返回一个对象引用，因此您可以使用 `new` 返回的值来调用新对象的方法：

```text
new Rectangle(100, 50).getArea()
```

表达式 `new Rectangle(100, 50)` 返回一个指向 Rectangle 对象的对象引用。如上所示，您可以使用点表示法来调用这个新 Rectangle 的 `getArea()` 方法，以计算新矩形的面积。

有些方法（例如 `getArea()`）会返回一个值。对于返回值的方法，您可以在表达式中使用该方法的调用。您可以将返回值赋给一个变量、用它来做出决策，或用它控制循环。下面的代码将 `getArea()` 返回的值赋给变量 `areaOfRectangle`：

```java
int areaOfRectangle = new Rectangle(100, 50).getArea();
```

请记住，在某个特定对象上调用方法，就等同于向该对象发送一条消息。在本例中，调用 `getArea()` 的对象就是构造方法所返回的矩形。

## 垃圾回收器

一些面向对象的语言要求您跟踪自己创建的所有对象，并在不再需要它们时显式地销毁它们。显式地管理内存既繁琐又容易出错。Java 平台允许您创建任意多的对象（当然，受限于您的系统能力），而您不必担心销毁它们。Java 运行时环境会在判定对象不再被使用时将其删除。这个过程被称为*垃圾回收(garbage collection)*。

当一个对象不再有任何引用指向它时，它就符合垃圾回收的条件。保存在变量中的引用通常在该变量超出作用域时被丢弃。或者，您也可以通过将变量设置为特殊值 null 来显式丢弃一个对象引用。请记住，一个程序可以拥有指向同一对象的多个引用；一个对象必须在所有指向它的引用都被丢弃之后，才符合垃圾回收的条件。

Java 运行时环境有一个垃圾回收器，它会定期释放不再被引用的对象所占用的内存。垃圾回收器会在判定时机合适时自动完成其工作。
