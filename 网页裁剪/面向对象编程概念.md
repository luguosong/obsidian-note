## 课程：面向对象编程概念

如果您以前从未使用过面向对象编程(object-oriented programming)语言，那么在开始编写任何代码之前，您需要先学习一些基本概念。本课程将向您介绍对象、类、继承、接口和包。每一部分的讨论都聚焦于这些概念如何与现实世界相关联，同时介绍 Java 编程语言的语法。

## 什么是对象？

对象是相关状态与行为的软件集合。软件对象常被用来模拟您在日常生活中遇到的真实世界对象。本节解释状态和行为如何在对象内部表示，介绍数据封装(data encapsulation)的概念，并解释以这种方式设计软件的好处。

## 什么是类？

类是创建对象的蓝图(blueprint)或原型(prototype)。本节定义一个模拟真实世界对象的状态和行为的类。它有意聚焦于基础知识，展示即使是一个简单的类也能清晰地建模状态和行为。

## 什么是继承？

继承为组织和构建软件提供了一种强大而自然的机制。本节解释类如何从其超类(superclass)继承状态和行为，并解释如何使用 Java 编程语言提供的简单语法从一个类派生出另一个类。

## 什么是接口？

接口是类与外部世界之间的一种契约。当一个类实现(implements)某个接口时，它就承诺提供该接口所公布的行为。本节定义一个简单的接口，并解释实现该接口的任何类所需做的必要更改。

## 什么是包？

包是一种以逻辑方式组织类和接口的命名空间(namespace)。将代码放入包中可使大型软件项目更易于管理。本节解释为什么这很有用，并向您介绍 Java 平台提供的应用程序编程接口(Application Programming Interface, API)。
