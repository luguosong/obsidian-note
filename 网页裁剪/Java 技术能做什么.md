## Java 技术能做什么？

通用的、高级的 Java 编程语言是一个功能强大的软件平台。Java 平台的每个完整实现都为你提供以下特性：

- **开发工具（Development Tools）**：开发工具提供了编译、运行、监控、调试以及为应用程序编写文档所需的一切。作为一名新开发者，你将使用的主要工具是 `javac` 编译器、`java` 启动器以及 `javadoc` 文档工具。
- **应用程序编程接口（API）**：API 提供了 Java 编程语言的核心功能。它提供了一大批可供你直接在自己应用程序中使用的实用类，涵盖从基本对象、网络与安全，到 XML 生成和数据库访问等方方面面。核心 API 非常庞大；若要概览其内容，请参阅 [Java Platform Standard Edition 8 Documentation](https://docs.oracle.com/javase/8/docs/index.html)。
- **部署技术（Deployment Technologies）**：JDK 软件提供了将应用程序部署给最终用户的标准机制，例如 Java Web Start 软件和 Java Plug-In 软件。
- **用户界面工具包（User Interface Toolkits）**：JavaFX、Swing 和 Java 2D 工具包使得创建复杂的图形用户界面（GUI）成为可能。
- **集成类库（Integration Libraries）**：诸如 Java IDL API、JDBC API、Java 命名与目录接口（JNDI）API、Java RMI，以及基于 Internet Inter-ORB 协议的 Java 远程方法调用技术（Java RMI-IIOP Technology）等集成类库，使得访问数据库并操纵远程对象成为可能。
