## 学习路径：2D 图形

本学习路径向您介绍 Java 2D™ API，并展示如何在 Java 程序中显示和打印 2D 图形。本学习路径面向希望丰富 Java 2D API 知识的开发者以及计算机图形学初学者。几乎每一节都包含相关示例来说明具体功能。Java 2D API 使您能够轻松执行以下任务：

- 绘制直线、矩形和任何其它几何形状。
- 用纯色或渐变及纹理填充这些形状。
- 绘制文本，并提供对字体和渲染过程的精细控制选项。
- 绘制图像，可选择性地应用过滤操作。
- 在上述任何渲染操作中应用合成（compositing）和变换（transforming）等操作。

本章还解释了诸如合成（compositing）等较少为人知的概念。

本章描述了在屏幕图像和屏幕外图像以及表面和打印设备上绘制的概念。本学习路径涵盖 Java 2D API 最常见的用法，并简要描述一些更高级的功能。

[**Java 2D 图形 API 概述（Overview of the Java 2D Graphics API）**](https://docs.oracle.com/javase/tutorial/2d/overview/index.html) 介绍关键的 Java 2D 概念，并描述 Java 2D 渲染模型。本课程比本学习路径的其它课程更具概念性，让您深入了解基本概念和类的描述。

[**图形入门（Getting started with Graphics）**](https://docs.oracle.com/javase/tutorial/2d/basic2d/index.html) 使用一个开发好的示例向您展示如何获取 Graphics 对象并将其用于常见的图形渲染任务。

[**处理几何图形（Working with Geometry）**](https://docs.oracle.com/javase/tutorial/2d/geometry/index.html) 教您如何使用 API 绘制图形图元和任意形状，以及如何应用精美的描边和填充样式。

[**使用文本 API（Working with Text APIs）**](https://docs.oracle.com/javase/tutorial/2d/text/index.html) 展示如何有效地使用文本 API，包括如何创建具有所需属性的 `Font` 对象、测量文本以及确定系统上可用字体的名称。

[**处理图像（Working with Images）**](https://docs.oracle.com/javase/tutorial/2d/images/index.html) 解释如何创建 BufferedImage 对象、执行图像过滤操作以及在图像上绘制。

[**打印（Printing）**](https://docs.oracle.com/javase/tutorial/2d/printing/index.html) 教您如何将 2D 图形渲染到打印机、打印复杂文档以及使用打印服务（Print Services）。

[**Java 2D 的高级主题（Advanced topics in Java 2D）**](https://docs.oracle.com/javase/tutorial/2d/advanced/index.html) 解释如何执行变换、裁剪绘制区域、合成重叠图形、指定渲染偏好以及控制渲染质量。
