## 什么是继承？

不同种类的对象之间往往存在一些共同之处。例如，山地自行车、公路自行车和双人自行车都具备自行车的共同特征（当前速度、当前踏频、当前档位）。然而，每种自行车还定义了使它们彼此不同的附加特征：双人自行车有两个座位和两套车把；公路自行车配有下弯式车把；一些山地自行车多了一个链轮，使它们拥有更低的齿比。

面向对象编程允许类从其他类*继承(inherit)*常用的状态和行为。在这个例子中，`Bicycle` 成为了 `MountainBike`、`RoadBike` 和 `TandemBike` 的*超类(superclass)*。在 Java 编程语言中，每个类只允许有一个直接超类，而每个超类则可能拥有数量不限的*子类(subclass)*：

![[oop-concepts-concepts-bikeHierarchy.gif]]

自行车类的层次结构。

创建子类的语法很简单。在类声明的开头，使用 `extends` 关键字，后接要继承的类的名称：

```
class MountainBike extends Bicycle {

    // 定义山地自行车的
    // 新字段和方法将写在这里

}
```

这使 `MountainBike` 拥有了与 `Bicycle` 相同的全部字段和方法，同时允许其代码专注于使其独具特色的那些功能。这让您的子类代码易于阅读。不过，您必须仔细记录每个超类所定义的状态和行为，因为这些代码并不会出现在每个子类的源文件中。
