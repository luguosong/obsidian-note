## 泛型的问题与练习

1. 编写一个泛型方法，统计集合中具有特定属性的元素个数（例如奇数整数、素数、回文）。
2. 下面的类能否编译？如果不能，为什么？
	```java
	public final class Algorithm {
	    public static <T> T max(T x, T y) {
	        return x > y ? x : y;
	    }
	}
	```
3. 编写一个泛型方法，交换数组中两个不同元素的位置。
4. 如果编译器在编译时擦除了所有类型参数，为什么你还应该使用泛型？
5. 下面的类在类型擦除之后会变成什么？
	```java
	public class Pair<K, V> {
	    public Pair(K key, V value) {
	        this.key = key;
	        this.value = value;
	    }
	    public K getKey() { return key; }
	    public V getValue() { return value; }
	    public void setKey(K key)     { this.key = key; }
	    public void setValue(V value) { this.value = value; }
	    private K key;
	    private V value;
	}
	```
6. 下面的方法在类型擦除之后会变成什么？
	```java
	public static <T extends Comparable<T>>
	    int findFirstGreaterThan(T[] at, T elem) {
	    // ...
	}
	```
7. 下面的方法能否编译？如果不能，为什么？
	```java
	public static void print(List<? extends Number> list) {
	    for (Number n : list)
	        System.out.print(n + " ");
	    System.out.println();
	}
	```
8. 编写一个泛型方法，找出列表在 \[begin, end) 范围内的最大元素。
9. 下面的类能否编译？如果不能，为什么？
	```java
	public class Singleton<T> {
	    public static T getInstance() {
	        if (instance == null)
	            instance = new Singleton<T>();
	        return instance;
	    }
	    private static T instance = null;
	}
	```
10. 给定下面的类：
	```java
	class Shape { /* ... */ }
	class Circle extends Shape { /* ... */ }
	class Rectangle extends Shape { /* ... */ }
	class Node<T> { /* ... */ }
	```
	下面的代码能否编译？如果不能，为什么？
	```java
	Node<Circle> nc = new Node<>();
	Node<Shape>  ns = nc;
	```
11. 考虑这个类：
	```java
	class Node<T> implements Comparable<T> {
	    public int compareTo(T obj) { /* ... */ }
	    // ...
	}
	```
	下面的代码能否编译？如果不能，为什么？
	```java
	Node<String> node = new Node<>();
	Comparable<String> comp = node;
	```
12. 你如何调用下面的方法，以找出列表中第一个与一组指定整数互素的整数？
	```java
	public static <T>
	    int findFirst(List<T> list, int begin, int end, UnaryPredicate<T> p)
	```
	注意，两个整数 *a* 和 *b* 互素，如果 gcd(*a, b*) = 1，其中 gcd 是最大公约数（greatest common divisor）的缩写。
