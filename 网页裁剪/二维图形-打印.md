## 课程：打印

由于 Java 2D API 使你能在任何表面上绘图，因此它的一个自然扩展是能够打印 Java 2D 图形。打印机可以被视为像显示器一样的图形设备。

Java 2D 打印 API 不仅限于打印图形。它还使你能打印应用程序用户界面的内容。内容可以通过在 Java 2D 打印 API 的格式控制下向打印机发送原始数据来打印，或通过使用 Java 2D 图形 API 来打印。

在本课中，你将探索 Java 2D 打印 API 的打印机和作业控制功能，它们是渲染元素的补充。你将学习如何查找系统或网络上配置的打印机，并发现有关这些打印机的信息（如它支持的纸张大小），以及为打印选择这些属性和用户对话框。

涉及打印的主要类和接口在 [`java.awt.print`](https://docs.oracle.com/javase/8/docs/api/java/awt/print/package-frame.html) 和 [`javax.print`](https://docs.oracle.com/javase/8/docs/api/javax/print/package-frame.html) 包中表示（后一个包使你能访问打印服务）。
