## 运算符问题与练习

## 问题

1. 考虑下面的代码片段。
	```text
	arrayOfInts[j] > arrayOfInts[j+1]
	```
	该代码包含哪些运算符？
2. 考虑下面的代码片段。
	```java
	int i = 10;
	int n = i++%5;
	```
	1. 代码执行后 `i` 和 `n` 的值各是多少？
	2. 如果不使用后缀递增运算符（`i++`），而是使用前缀版本（`++i`），`i` 和 `n` 的最终值各是多少？
3. 要反转 `boolean` 的值，您会使用哪个运算符？
4. 哪个运算符用于比较两个值，`=` 还是 `==`？
5. 解释下面的代码示例：`result = someCondition ? value1 : value2;`

## 练习

1. 修改下面的程序，使用复合赋值：
	```java
	class ArithmeticDemo {
	     public static void main (String[] args){
	          
	          int result = 1 + 2; // result 现在是 3
	          System.out.println(result);
	          result = result - 1; // result 现在是 2
	          System.out.println(result);
	          result = result * 2; // result 现在是 4
	          System.out.println(result);
	          result = result / 2; // result 现在是 2
	          System.out.println(result);
	          result = result + 8; // result 现在是 10
	          result = result % 7; // result 现在是 3
	          System.out.println(result);
	     }
	}
	```
2. 在下面的程序中，解释为什么值 "6" 会被连续打印两次：
	```java
	class PrePostDemo {
	    public static void main(String[] args){
	        int i = 3;
	        i++;
	        System.out.println(i);    // "4"
	        ++i;                     
	        System.out.println(i);    // "5"
	        System.out.println(++i);  // "6"
	        System.out.println(i++);  // "6"
	        System.out.println(i);    // "7"
	    }
	}
	```
