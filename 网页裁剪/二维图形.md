## 路线：二维图形

本路线向你介绍 Java 2D™ API，并向你展示如何在 Java 程序中显示和打印二维图形。本路线面向希望丰富 Java 2D API 知识的开发者以及计算机图形学的初学者。几乎每一节都包含相关示例以说明特定功能。Java 2D API 使你能轻松执行以下任务：

- 绘制直线、矩形和任何其他几何形状。
- 用纯色或渐变与纹理填充这些形状。
- 绘制文本，并具有对字体和渲染过程的精细控制选项。
- 绘制图像，可选地应用过滤操作。
- 在上述任何渲染操作期间应用诸如合成和变换之类的操作。

本章还解释了合成等不太为人所知的概念。

本章描述了在屏幕和屏幕外图像以及表面和打印机设备上绘制的概念。本路线涵盖 Java 2D API 最常见的用法，并简要描述一些更高级的特性。

[[二维图形-API概述|**Java 2D 图形 API 概述**]] 介绍关键的 Java 2D 概念并描述 Java 2D 渲染模型。本课比本路线的其他课程更具概念性，它使你能深入了解基本概念和类描述。

[[二维图形-图形入门|**图形入门**]] 使用一个开发的示例向你展示如何获取 Graphics 对象并将其用于常见的图形渲染任务。

[[二维图形-几何图形|**使用几何图形**]] 教你如何使用 API 绘制图形原语和任意形状，以及如何应用花式描边和填充样式。

[[二维图形-文本|**使用文本 API**]] 向你展示如何有效地使用文本 API，包括如何创建具有所需属性的 `Font` 对象、测量文本，以及确定系统上可用字体的名称。

[[二维图形-图像|**使用图像**]] 解释如何创建 BufferedImage 对象、执行图像过滤操作以及在图像上绘制。

[[二维图形-打印|**打印**]] 教你如何将二维图形渲染到打印机、打印复杂文档以及使用打印服务。

[[二维图形-高级主题|**Java 2D 的高级主题**]] 解释如何执行变换、裁剪绘制区域、合成重叠图形、指定渲染偏好以及控制渲染质量。
