## 字符串

字符串在 Java 编程中被广泛使用，它是一系列字符。在 Java 编程语言中，字符串是对象。

Java 平台提供了 [`String`](https://docs.oracle.com/javase/8/docs/api/java/lang/String.html) 类来创建和操作字符串。

## 创建字符串

创建字符串最直接的方式是编写：

```java
String greeting = "Hello world!";
```

在这种情况下，"Hello world!" 是一个*字符串字面量（string literal）*——即你代码中被双引号括起来的一系列字符。每当它在你的代码中遇到一个字符串字面量，编译器就会创建一个具有其值的 `String` 对象——在本例中为 `Hello world!`。

与任何其他对象一样，你可以使用 `new` 关键字和构造方法来创建 `String` 对象。`String` 类有十三个构造方法，允许你使用不同的来源提供字符串的初始值，例如一个字符数组：

```java
char[] helloArray = { 'h', 'e', 'l', 'l', 'o', '.' };
String helloString = new String(helloArray);
System.out.println(helloString);
```

这段代码片段的最后一行显示 `hello`。

---

**注意：** `String` 类是不可变的，因此一旦创建，`String` 对象就不能被更改。`String` 类有许多方法（其中一些将在下面讨论），它们看起来是在修改字符串。由于字符串是不可变的，这些方法实际做的是创建并返回一个包含操作结果的新字符串。

---

## 字符串长度

用于获取对象信息的方法被称为*访问器方法（accessor method）*。一个你可以用于字符串的访问器方法是 `length()` 方法，它返回字符串对象中包含的字符数。在下面两行代码执行之后，`len` 等于 17：

```java
String palindrome = "Dot saw I was Tod";
int len = palindrome.length();
```

*回文（palindrome）*是一个对称的单词或句子——忽略大小写和标点，它正读和反读拼写相同。下面是一个简短但低效的反转回文字符串的程序。它调用 `String` 方法 `charAt(i)`，该方法返回字符串中第 i <sup>个</sup> 字符（从 0 开始计数）。

```java
public class StringDemo {
    public static void main(String[] args) {
        String palindrome = "Dot saw I was Tod";
        int len = palindrome.length();
        char[] tempCharArray = new char[len];
        char[] charArray = new char[len];
        
        // 将原字符串放入一个
        // 字符数组
        for (int i = 0; i < len; i++) {
            tempCharArray[i] = 
                palindrome.charAt(i);
        } 
        
        // 反转字符数组
        for (int j = 0; j < len; j++) {
            charArray[j] =
                tempCharArray[len - 1 - j];
        }
        
        String reversePalindrome =
            new String(charArray);
        System.out.println(reversePalindrome);
    }
}
```

运行该程序会产生如下输出：

```text
doT saw I was toD
```

为了完成字符串反转，程序必须将字符串转换为字符数组（第一个 `for` 循环），将数组反转到第二个数组中（第二个 `for` 循环），然后再转换回字符串。[`String`](https://docs.oracle.com/javase/8/docs/api/java/lang/String.html) 类包含一个方法 `getChars()`，用于将字符串或字符串的一部分转换为字符数组，因此我们可以用

```java
palindrome.getChars(0, len, tempCharArray, 0);
```

来替换上面程序中的第一个 `for` 循环。

## 拼接字符串

`String` 类包含一个用于拼接两个字符串的方法：

```java
string1.concat(string2);
```

它返回一个新的字符串，即 string1 在末尾加上 string2。

你也可以对字符串字面量使用 `concat()` 方法，如：

```java
"My name is ".concat("Rumplestiltskin");
```

字符串更常见的是用 `+` 运算符来拼接，如

```java
"Hello," + " world" + "!"
```

结果为

```text
"Hello, world!"
```

`+` 运算符在 `print` 语句中被广泛使用。例如：

```java
String string1 = "saw I was ";
System.out.println("Dot " + string1 + "Tod");
```

打印出

```text
Dot saw I was Tod
```

这种拼接可以是任何对象的混合。对于每个非 `String` 的对象，都会调用其 `toString()` 方法将其转换为 `String`。

---

**注意：** Java 编程语言不允许字符串字面量在源文件中跨行，因此你必须在多行字符串每一行的末尾使用 `+` 拼接运算符。例如：

```java
String quote = 
    "Now is the time for all good " +
    "men to come to the aid of their country.";
```

使用 `+` 拼接运算符在行间断开字符串，在 `print` 语句中同样非常常见。

---

## 创建格式化字符串

你已经见过使用 `printf()` 和 `format()` 方法来打印带有格式化数字的输出。`String` 类有一个等价的类方法 `format()`，它返回一个 `String` 对象而非 `PrintStream` 对象。

使用 `String` 的静态 `format()` 方法使你能够创建一个可重用的格式化字符串，而非一次性打印语句。例如，与其使用

```java
System.out.printf("The value of the float " +
                  "variable is %f, while " +
                  "the value of the " + 
                  "integer variable is %d, " +
                  "and the string is %s", 
                  floatVar, intVar, stringVar);
```

你可以编写

```java
String fs;
fs = String.format("The value of the float " +
                   "variable is %f, while " +
                   "the value of the " + 
                   "integer variable is %d, " +
                   " and the string is %s",
                   floatVar, intVar, stringVar);
System.out.println(fs);
```
