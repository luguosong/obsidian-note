## 枚举类型

*枚举类型(enum type)*是一种特殊的数据类型，它使一个变量可以成为一组预定义常量(predefined constant)中的一员。该变量必须等于为其预定义的某个值。常见的例子包括罗盘方向（NORTH、SOUTH、EAST 和 WEST 这些值）以及星期几。

由于它们是常量，枚举类型字段的名称使用大写字母。

在 Java 编程语言中，您可以使用 `enum` 关键字来定义枚举类型。例如，您可以这样指定一个表示星期几的枚举类型：

```java
public enum Day {
    SUNDAY, MONDAY, TUESDAY, WEDNESDAY,
    THURSDAY, FRIDAY, SATURDAY 
}
```

每当您需要表示一组固定的常量时，都应该使用枚举类型。这包括天然适合用枚举表示的类型（如太阳系中的行星），以及在编译时就能知道所有可能取值的数据集合——例如菜单上的选项、命令行标志等等。

下面是一段代码，展示了如何使用上面定义的 `Day` 枚举：

```java
public class EnumTest {
    Day day;
    
    public EnumTest(Day day) {
        this.day = day;
    }
    
    public void tellItLikeItIs() {
        switch (day) {
            case MONDAY:
                System.out.println("Mondays are bad.");
                break;
                    
            case FRIDAY:
                System.out.println("Fridays are better.");
                break;
                         
            case SATURDAY: case SUNDAY:
                System.out.println("Weekends are best.");
                break;
                        
            default:
                System.out.println("Midweek days are so-so.");
                break;
        }
    }
    
    public static void main(String[] args) {
        EnumTest firstDay = new EnumTest(Day.MONDAY);
        firstDay.tellItLikeItIs();
        EnumTest thirdDay = new EnumTest(Day.WEDNESDAY);
        thirdDay.tellItLikeItIs();
        EnumTest fifthDay = new EnumTest(Day.FRIDAY);
        fifthDay.tellItLikeItIs();
        EnumTest sixthDay = new EnumTest(Day.SATURDAY);
        sixthDay.tellItLikeItIs();
        EnumTest seventhDay = new EnumTest(Day.SUNDAY);
        seventhDay.tellItLikeItIs();
    }
}
```

输出为：

```text
Mondays are bad.
Midweek days are so-so.
Fridays are better.
Weekends are best.
Weekends are best.
```

Java 编程语言的枚举类型比其它语言中的对应功能要强大得多。`enum` 声明会定义一个*类*（称为*枚举类型*）。枚举类体可以包含方法和其它字段。编译器在创建枚举时会自动添加一些特殊的方法。例如，它们有一个静态的 `values` 方法，该方法返回一个数组，其中包含枚举的所有值，顺序与声明时的顺序一致。此方法通常与 for-each 构造结合使用，用于遍历枚举类型的值。例如，下面来自 `Planet` 类示例的代码会遍历太阳系中的所有行星。

```java
for (Planet p : Planet.values()) {
    System.out.printf("Your weight on %s is %f%n",
                      p, p.surfaceWeight(mass));
}
```

---

**注意：** *所有*枚举都隐式继承自 `java.lang.Enum`。由于一个类只能继承一个父类（参见 [声明类（Declaring Classes）](https://docs.oracle.com/javase/tutorial/java/javaOO/classdecl.html)），Java 语言不支持状态的多重继承(multiple inheritance)（参见 [状态、实现和类型的多重继承（Multiple Inheritance of State, Implementation, and Type）](https://docs.oracle.com/javase/tutorial/java/IandI/multipleinheritance.html)），因此枚举不能再继承其它任何类。

---

在下面的示例中，`Planet` 是一个表示太阳系中行星的枚举类型。它们用恒定的质量和半径属性来定义。

每个枚举常量在声明时都带有 mass 和 radius 参数的值。这些值在创建常量时传递给构造方法。Java 要求常量必须先于任何字段或方法定义。此外，当存在字段和方法时，枚举常量列表必须以分号结尾。

---

**注意：** 枚举类型的构造方法必须是包级私有(package-private)或私有(private)访问级别。它会自动创建在枚举体开头定义的常量。您不能自行调用枚举构造方法。

---

除了属性和构造方法之外，`Planet` 还有允许您获取每个行星上的表面重力和物体重量的方法。下面是一个示例程序，它接收您在地球上的体重（任意单位），并计算和打印您在所有行星上的体重（同一单位）：

```java
public enum Planet {
    MERCURY (3.303e+23, 2.4397e6),
    VENUS   (4.869e+24, 6.0518e6),
    EARTH   (5.976e+24, 6.37814e6),
    MARS    (6.421e+23, 3.3972e6),
    JUPITER (1.9e+27,   7.1492e7),
    SATURN  (5.688e+26, 6.0268e7),
    URANUS  (8.686e+25, 2.5559e7),
    NEPTUNE (1.024e+26, 2.4746e7);

    private final double mass;   // in kilograms
    private final double radius; // in meters
    Planet(double mass, double radius) {
        this.mass = mass;
        this.radius = radius;
    }
    private double mass() { return mass; }
    private double radius() { return radius; }

    // universal gravitational constant  (m3 kg-1 s-2)
    public static final double G = 6.67300E-11;

    double surfaceGravity() {
        return G * mass / (radius * radius);
    }
    double surfaceWeight(double otherMass) {
        return otherMass * surfaceGravity();
    }
    public static void main(String[] args) {
        if (args.length != 1) {
            System.err.println("Usage: java Planet <earth_weight>");
            System.exit(-1);
        }
        double earthWeight = Double.parseDouble(args[0]);
        double mass = earthWeight/EARTH.surfaceGravity();
        for (Planet p : Planet.values())
           System.out.printf("Your weight on %s is %f%n",
                             p, p.surfaceWeight(mass));
    }
}
```

如果您从命令行运行 `Planet.class` 并传入参数 175，会得到如下输出：

```bash
$ java Planet 175
Your weight on MERCURY is 66.107583
Your weight on VENUS is 158.374842
Your weight on EARTH is 175.000000
Your weight on MARS is 66.279007
Your weight on JUPITER is 442.847567
Your weight on SATURN is 186.552719
Your weight on URANUS is 158.397260
Your weight on NEPTUNE is 199.207413
```
