## 问题与练习的答案：面向对象编程概念

## 问题的答案

1. 真实世界的对象包含 **状态(state)** 和 **行为(behavior)**。
2. 软件对象的状态存储在 **字段(field)** 中。
3. 软件对象的行为通过 **方法(method)** 来展现。
4. 将内部数据对外部世界隐藏，仅通过公开暴露的方法访问它，这种方式被称为数据 **封装(encapsulation)**。
5. 软件对象的图纸被称为 **类(class)**。
6. 公共行为可以在一个 **超类(superclass)** 中定义，并使用 **extends** 关键字继承到一个 **子类(subclass)** 中。
7. 一组没有实现的方法集合被称为 **接口(interface)**。
8. 按功能组织类和接口的命名空间被称为 **包(package)**。
9. 术语 API 代表 **应用程序编程接口(Application Programming Interface)**。
