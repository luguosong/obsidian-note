## 接口的问题与练习

## 问题

1. 一个实现了 `java.lang.CharSequence` 接口的类，必须实现哪些方法？
2. 下面的接口有什么问题？
	```java
	public interface SomethingIsWrong {
	    void aMethod(int aValue){
	        System.out.println("Hi Mom");
	    }
	}
	```
3. 修复问题 2 中的接口。
4. 下面的接口是否合法？
	```java
	public interface Marker {
	}
	```

## 练习

1. 编写一个实现 `java.lang` 包中 `CharSequence` 接口的类。你的实现应当返回反转后的字符串。从本书中选取一句话作为数据。编写一个小的 `main` 方法来测试你的类；确保调用全部四个方法。
2. 假设你编写了一个时间服务器，定期向其客户端通知当前的日期和时间。编写一个接口，服务器可以用它来对客户端强制实施特定的协议。
