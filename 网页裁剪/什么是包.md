## 什么是包？

包(package)是一个用于组织一组相关类和接口的命名空间(namespace)。从概念上讲，您可以把包想象成计算机上不同的文件夹。您可能把 HTML 页面放在一个文件夹里，图片放在另一个文件夹里，而脚本或应用程序放在又一个文件夹里。由于用 Java 编程语言编写的软件可能由成百上千个独立的类组成，因此将相关的类和接口放入包中是有意义的组织方式。

Java 平台提供了一个庞大的类库（一组包），适合在您自己的应用程序中使用。这个库被称为「应用程序编程接口(Application Programming Interface)」，简称「API」。它的各个包代表了通用编程中最常见的那些任务。例如，一个 `String` 对象包含字符字符串的状态和行为；一个 `File` 对象让程序员能够轻松地创建、删除、检查、比较或修改文件系统上的文件；一个 `Socket` 对象允许创建和使用网络套接字；各种图形用户界面(GUI)对象则控制着按钮、复选框以及与图形用户界面相关的任何其他元素。可供选择的类数以千计。这使您——程序员——能够专注于自己特定应用程序的设计，而无需关心让它运行所需的底层基础设施。

[Java 平台 API 规范（Java Platform API Specification）](https://docs.oracle.com/javase/8/docs/api/index.html)包含了 Java SE 平台所提供的全部包、接口、类、字段和方法的完整清单。请在浏览器中打开该页面并加入书签。作为程序员，它将成为您最重要的参考文档。
