## 课程：语言基础

## 变量

你已经了解到对象将其状态存储在字段中。然而，Java 编程语言也使用"变量"这一术语。本节讨论这种关系，以及变量命名规则和约定、基本数据类型（原始类型、字符串和数组）、默认值和字面量。

## 运算符

本节描述 Java 编程语言的运算符。它首先介绍最常用的运算符，最后介绍较少使用的运算符。每个讨论都包含你可以编译和运行的代码示例。

## 表达式、语句和块

运算符可用于构建计算值的表达式；表达式是语句的核心组件；语句可以分组为块。本节使用你已经看过的示例代码讨论表达式、语句和块。

## 控制流语句

本节描述 Java 编程语言支持的控制流语句。它涵盖决策、循环和分支语句，使你的程序能够有条件地执行特定的代码块。
