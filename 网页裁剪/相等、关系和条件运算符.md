## 相等、关系和条件运算符

## 相等和关系运算符

相等和关系运算符用于确定一个操作数是否大于、小于、等于或不等于另一个操作数。这些运算符中的大多数您可能也很熟悉。请记住，在测试两个基本类型(primitive)的值是否相等时，必须使用「`==`」而不是「`=`」。

```bash
==      等于
!=      不等于
>       大于
>=      大于等于
<       小于
<=      小于等于
```

下面的程序 [`ComparisonDemo`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/ComparisonDemo.java) 测试了比较运算符：

```java
class ComparisonDemo {

    public static void main(String[] args){
        int value1 = 1;
        int value2 = 2;
        if(value1 == value2)
            System.out.println("value1 == value2");
        if(value1 != value2)
            System.out.println("value1 != value2");
        if(value1 > value2)
            System.out.println("value1 > value2");
        if(value1 < value2)
            System.out.println("value1 < value2");
        if(value1 <= value2)
            System.out.println("value1 <= value2");
    }
}
```

输出：

```text
value1 != value2
value1 <  value2
value1 <= value2
```

## 条件运算符

`&&` 和 `||` 运算符对两个布尔表达式执行*条件与(Conditional-AND)*和*条件或(Conditional-OR)*操作。这些运算符表现出「短路(short-circuiting)」行为，即只在需要时才对第二个操作数求值。

```text
&& 条件与
|| 条件或
```

下面的程序 [`ConditionalDemo1`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/ConditionalDemo1.java) 测试了这些运算符：

```java
class ConditionalDemo1 {

    public static void main(String[] args){
        int value1 = 1;
        int value2 = 2;
        if((value1 == 1) && (value2 == 2))
            System.out.println("value1 is 1 AND value2 is 2");
        if((value1 == 1) || (value2 == 1))
            System.out.println("value1 is 1 OR value2 is 1");
    }
}
```

另一个条件运算符是 `?:`，它可以看作是 `if-then-else` 语句的简写形式（在本课的[控制流语句（Control Flow Statements）](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/flow.html)一节中讨论）。该运算符也被称为*三元运算符(ternary operator)*，因为它使用三个操作数。在下面的示例中，该运算符应读作：「如果 `someCondition` 为 `true`，则将 `value1` 的值赋给 `result`。否则，将 `value2` 的值赋给 `result`。」

下面的程序 [`ConditionalDemo2`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/ConditionalDemo2.java) 测试了 `?:` 运算符：

```java
class ConditionalDemo2 {

    public static void main(String[] args){
        int value1 = 1;
        int value2 = 2;
        int result;
        boolean someCondition = true;
        result = someCondition ? value1 : value2;

        System.out.println(result);
    }
}
```

由于 `someCondition` 为 true，该程序向屏幕打印 "1"。如果使用 `?:` 运算符能使代码更具可读性，则用它代替 `if-then-else` 语句；例如，当表达式很简洁且没有副作用（如赋值）时。

## 类型比较运算符 instanceof

`instanceof` 运算符将一个对象与指定类型进行比较。您可以用它来测试一个对象是否是某个类的实例、某个子类的实例，或是实现了某个特定接口的类的实例。

下面的程序 [`InstanceofDemo`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/InstanceofDemo.java) 定义了一个父类（名为 `Parent`）、一个简单接口（名为 `MyInterface`），以及一个继承自父类并实现该接口的子类（名为 `Child`）。

```java
class InstanceofDemo {
    public static void main(String[] args) {

        Parent obj1 = new Parent();
        Parent obj2 = new Child();

        System.out.println("obj1 instanceof Parent: "
            + (obj1 instanceof Parent));
        System.out.println("obj1 instanceof Child: "
            + (obj1 instanceof Child));
        System.out.println("obj1 instanceof MyInterface: "
            + (obj1 instanceof MyInterface));
        System.out.println("obj2 instanceof Parent: "
            + (obj2 instanceof Parent));
        System.out.println("obj2 instanceof Child: "
            + (obj2 instanceof Child));
        System.out.println("obj2 instanceof MyInterface: "
            + (obj2 instanceof MyInterface));
    }
}

class Parent {}
class Child extends Parent implements MyInterface {}
interface MyInterface {}
```

输出：

```text
obj1 instanceof Parent: true
obj1 instanceof Child: false
obj1 instanceof MyInterface: false
obj2 instanceof Parent: true
obj2 instanceof Child: true
obj2 instanceof MyInterface: true
```

使用 `instanceof` 运算符时，请记住 `null` 不是任何对象的实例。
