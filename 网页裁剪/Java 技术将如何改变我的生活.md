## Java 技术将如何改变我的生活？

如果你学习了 Java 编程语言，我们无法向你承诺名望、财富，甚至一份工作。不过，它很可能会让你的程序变得更好，并且比使用其他语言需要更少的精力。我们相信 Java 技术将帮助你做到以下事情：

- **快速上手（Get started quickly）**：尽管 Java 编程语言是一种功能强大的面向对象语言，但它易于学习，尤其对于已经熟悉 C 或 C++ 的程序员而言。
- **编写更少的代码（Write less code）**：程序度量指标（类的数量、方法的数量等）的对比表明，用 Java 编程语言编写的程序，其规模可能比用 C++ 编写的相同程序小四倍。
- **编写更优质的代码（Write better code）**：Java 编程语言鼓励良好的编码实践，自动垃圾回收机制帮助你避免内存泄漏。其面向对象特性、JavaBeans™ 组件架构，以及广泛且易于扩展的 API，让你能够重用现有、经过测试的代码，从而引入更少的缺陷。
- **更快速地开发程序（Develop programs more quickly）**：Java 编程语言比 C++ 更简单，因此用它编写代码时，开发时间可能快达两倍。你的程序也将需要更少的代码行。
- **避免平台依赖（Avoid platform dependencies）**：通过避免使用以其他语言编写的库，你可以保持程序的可移植性。
- **一次编写，到处运行（Write once, run anywhere）**：由于用 Java 编程语言编写的应用程序会被编译成与机器无关的字节码，它们可以在任何 Java 平台上一致地运行。
- **更轻松地分发软件（Distribute software more easily）**：借助 Java Web Start 软件，用户只需单击鼠标即可启动你的应用程序。启动时的自动版本检查可确保用户始终使用你软件的最新版本。如果有可用更新，Java Web Start 软件将自动更新其安装。
