本 Java 教程是面向 JDK 8 编写的。本页所述的示例与实践并未利用后续版本中引入的改进，且可能使用了现已不再可用的技术。
如需利用最新版本改进的更新版教程，请参见 [Dev.java](https://dev.java/learn/)。
如需了解 Java SE 9 及后续各版本中语言特性的更新摘要，请参见 [Java 语言变更（Java Language Changes）](https://docs.oracle.com/pls/topic/lookup?ctx=en/java/javase&id=java_language_changes)。
如需了解所有 JDK 版本的新特性、增强功能以及被移除或弃用的选项，请参见 [JDK 发行说明（JDK Release Notes）](https://www.oracle.com/technetwork/java/javase/jdk-relnotes-index-2162236.html)。

## 学习路径：入门

本学习路径提供了开始使用 Java 编程语言所需了解的全部内容。

[[Java 技术现象|**Java 技术现象（The Java Technology Phenomenon）**]] 概述 Java 技术的整体情况。它同时讨论了 Java 编程语言和 Java 平台，对该技术能做什么以及如何让您的开发更轻松提供了全面的概述。

[[Hello World 应用程序|**「Hello World!」应用程序（The "Hello World!" Application）**]] 本动手实践教程描述了创建一个简单的「Hello World!」应用程序需要下载什么、安装什么以及输入什么。它为 NetBeans™ 集成开发环境（NetBeans IDE）、Microsoft Windows、Solaris™ 操作系统（Solaris OS）、Linux 和 Mac 用户分别提供了不同的操作说明。

[[深入剖析 Hello World|**深入剖析「Hello World!」（A Closer Look at "Hello World!"）**]] 讨论「Hello World!」应用程序，详细描述代码的每一部分。内容涵盖源代码注释、`HelloWorldApp` 类定义块以及 `main` 方法。

[[常见问题及其解决方案|**常见问题及其解决方案（Common Problems (and Their Solutions)）**]] 如果您在本学习路径中编译或运行程序时遇到困难，请查阅此处。
