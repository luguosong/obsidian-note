## 运算符

既然您已经学会了如何声明和初始化变量，您大概想知道如何用它们来*做点什么*。学习 Java 编程语言的运算符(operator)是一个不错的起点。运算符是对一个、两个或三个*操作数(operand)*执行特定操作并返回结果的特殊符号。

在我们探索 Java 编程语言的运算符时，提前了解哪些运算符具有最高的优先级(precedence)可能会对您有所帮助。下表中的运算符按优先级顺序列出。运算符在表中越靠近顶部，其优先级越高。优先级较高的运算符会在优先级相对较低的运算符之前被求值。同一行上的运算符具有相同的优先级。当相同优先级的运算符出现在同一个表达式中时，必须有一条规则来决定哪个先被求值。除赋值运算符外，所有二元运算符(binary operator)都从左到右求值；赋值运算符则从右到左求值。

| 运算符 | 优先级 |
| --- | --- |
| 后置（postfix） | `*expr*++ *expr*--` |
| 一元（unary） | `++*expr* --*expr* +*expr* -*expr* ~ !` |
| 乘法（multiplicative） | `* / %` |
| 加法（additive） | `+ -` |
| 移位（shift） | `<< >> >>>` |
| 关系（relational） | `< > <= >= instanceof` |
| 相等（equality） | `== !=` |
| 按位与（bitwise AND） | `&` |
| 按位异或（bitwise exclusive OR） | `^` |
| 按位或（bitwise inclusive OR） | `\|` |
| 逻辑与（logical AND） | `&&` |
| 逻辑或（logical OR） | `\|\|` |
| 三元（ternary） | `? :` |
| 赋值（assignment） | `= += -= *= /= %= &= ^= \|= <<= >>= >>>=` |

在通用编程中，某些运算符出现的频率往往比其它运算符高；例如，赋值运算符「`=`」远比无符号右移运算符「`>>>`」常见。有鉴于此，下面的讨论首先聚焦于您最可能经常使用的运算符，最后才讨论那些不那么常见的运算符。每一部分的讨论都附有可编译和运行的示例代码。研究其输出将有助于巩固您刚刚学到的知识。
