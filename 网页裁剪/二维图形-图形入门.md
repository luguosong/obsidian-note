## 课程：图形入门

Java 2D API 功能强大且复杂。然而，Java 2D API 的绝大多数用途都使用了封装在 `java.awt.Graphics` 类中的小部分功能。本课涵盖应用程序开发者最常见的需求。较少见的需求在后面的 [[二维图形-高级主题|Java 2D API 高级主题]]中描述。

[`Graphics`](https://docs.oracle.com/javase/8/docs/api/java/awt/Graphics.html) 类的大多数方法可以分为两个基本组：

- 绘制和填充方法，使你能渲染基本形状、文本和图像
- 属性设置方法，影响绘制和填充的显示方式

`setFont` 和 `setColor` 等方法定义绘制和填充方法的渲染方式。

此图说明这些方法如何与图形对象相关：

![[2d-basic2d-graphicMethods.gif]]

绘制方法包括：

- `drawString` —— 用于绘制文本
	```java
	g.drawString("Hello", 10, 10);
- `drawImage` —— 用于绘制图像
	```java
	g.drawImage(img, 
	            0, 0, width, height,
	            0, 0, imageWidth, imageHeight,
	            null);
- `drawLine`、`drawArc`、`drawRect`、`drawOval`、`drawPolygon` —— 用于绘制几何形状
	```java
	g2.draw(new Line2D.Double(0, 0, 30, 40));

根据你当前的需求，你可以基于以下标准在 `Graphics` 类的多种方法中选择：

- 你是想以原始大小在指定位置渲染图像，还是缩放它以适合给定矩形
- 你是想用颜色填充图像的透明区域，还是保持它们透明

填充方法适用于几何形状，包括 `fillArc`、`fillRect`、`fillOval`、`fillPolygon`。

无论你绘制一行文本还是图像，请记住在 2D 图形中每个点都由其 x 和 y [[二维图形-坐标系|坐标]]确定。所有绘制和填充方法都需要此信息，它确定文本或图像应在何处渲染。

例如，要绘制一条直线，应用程序调用以下方法：

`java.awt.Graphics.drawLine(int x1, int y1, int x2, int y2)`

在此代码中，*(x1, y1)* 是直线的起点，*(x2, y2)* 是直线的终点。

因此绘制水平直线的代码如下：

`Graphics.drawLine(20, 100, 120, 100);`
