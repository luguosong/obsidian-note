## 分支语句

## break 语句

`break` 语句有两种形式：带标签(labeled)和不带标签(unlabeled)的。您在前面对 `switch` 语句的讨论中已经见过不带标签的形式。您也可以使用不带标签的 `break` 来终止 `for`、`while` 或 `do-while` 循环，如下面的 [`BreakDemo`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/BreakDemo.java) 程序所示：

```java
class BreakDemo {
    public static void main(String[] args) {

        int[] arrayOfInts = 
            { 32, 87, 3, 589,
              12, 1076, 2000,
              8, 622, 127 };
        int searchfor = 12;

        int i;
        boolean foundIt = false;

        for (i = 0; i < arrayOfInts.length; i++) {
            if (arrayOfInts[i] == searchfor) {
                foundIt = true;
                break;
            }
        }

        if (foundIt) {
            System.out.println("Found " + searchfor + " at index " + i);
        } else {
            System.out.println(searchfor + " not in the array");
        }
    }
}
```

该程序在一个数组中搜索数字 12。以粗体显示的 `break` 语句在该值被找到时终止 `for` 循环。随后控制流转到 `for` 循环之后的语句。该程序的输出为：

```text
Found 12 at index 4
```

不带标签的 `break` 语句终止最内层的 `switch`、`for`、`while` 或 `do-while` 语句，而带标签的 `break` 则终止外层的语句。下面的程序 [`BreakWithLabelDemo`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/BreakWithLabelDemo.java) 与前一个程序类似，但它使用嵌套的 `for` 循环在二维数组中搜索一个值。当找到该值时，带标签的 `break` 会终止外层的 `for` 循环（标签为「search」）：

```java
class BreakWithLabelDemo {
    public static void main(String[] args) {

        int[][] arrayOfInts = { 
            { 32, 87, 3, 589 },
            { 12, 1076, 2000, 8 },
            { 622, 127, 77, 955 }
        };
        int searchfor = 12;

        int i;
        int j = 0;
        boolean foundIt = false;

    search:
        for (i = 0; i < arrayOfInts.length; i++) {
            for (j = 0; j < arrayOfInts[i].length;
                 j++) {
                if (arrayOfInts[i][j] == searchfor) {
                    foundIt = true;
                    break search;
                }
            }
        }

        if (foundIt) {
            System.out.println("Found " + searchfor + " at " + i + ", " + j);
        } else {
            System.out.println(searchfor + " not in the array");
        }
    }
}
```

这是该程序的输出。

```text
Found 12 at 1, 0
```

`break` 语句终止带标签的语句；它并不把控制流转移到标签处。控制流被转移到紧跟在被终止的带标签语句之后的那条语句。

## continue 语句

`continue` 语句跳过 `for`、`while` 或 `do-while` 循环的当前迭代。不带标签的形式跳到最内层循环体的末尾，并对控制该循环的 `boolean` 表达式求值。下面的程序 [`ContinueDemo`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/ContinueDemo.java) 遍历一个 `String`，统计字母「p」出现的次数。如果当前字符不是 p，`continue` 语句就跳过循环的其余部分，继续处理下一个字符。如果它*是*「p」，程序就递增字母计数。

```java
class ContinueDemo {
    public static void main(String[] args) {

        String searchMe = "peter piper picked a " + "peck of pickled peppers";
        int max = searchMe.length();
        int numPs = 0;

        for (int i = 0; i < max; i++) {
            // interested only in p's
            if (searchMe.charAt(i) != 'p')
                continue;

            // process p's
            numPs++;
        }
        System.out.println("Found " + numPs + " p's in the string.");
    }
}
```

这是该程序的输出：

```text
Found 9 p's in the string.
```

为了更清楚地看到这一效果，试着去掉 `continue` 语句并重新编译。当您再次运行程序时，计数会出错，显示找到了 35 个 p 而不是 9 个。

带标签的 `continue` 语句跳过用给定标签标记的外层循环的当前迭代。下面的示例程序 [`ContinueWithLabelDemo`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/ContinueWithLabelDemo.java) 使用嵌套循环在另一个字符串中搜索某个子字符串。这里需要两层嵌套循环：一层用于遍历子字符串，另一层用于遍历被搜索的字符串。下面的程序 [`ContinueWithLabelDemo`](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/examples/ContinueWithLabelDemo.java) 使用带标签形式的 continue 来跳过外层循环中的一次迭代。

```java
class ContinueWithLabelDemo {
    public static void main(String[] args) {

        String searchMe = "Look for a substring in me";
        String substring = "sub";
        boolean foundIt = false;

        int max = searchMe.length() - 
                  substring.length();

    test:
        for (int i = 0; i <= max; i++) {
            int n = substring.length();
            int j = i;
            int k = 0;
            while (n-- != 0) {
                if (searchMe.charAt(j++) != substring.charAt(k++)) {
                    continue test;
                }
            }
            foundIt = true;
                break test;
        }
        System.out.println(foundIt ? "Found it" : "Didn't find it");
    }
}
```

这是该程序的输出。

```text
Found it
```

## return 语句

最后一种分支语句是 `return` 语句。`return` 语句退出当前方法，控制流返回到调用该方法的位置。`return` 语句有两种形式：一种返回一个值，另一种不返回值。要返回一个值，只需把该值（或一个计算该值的表达式）放在 `return` 关键字之后。

```text
return ++count;
```

返回值的数据类型必须与方法声明的返回值类型相匹配。当方法声明为 `void` 时，使用不返回值形式的 `return`。

```text
return;
```
