# markdown语法

## 基本语法

### 标题

```
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题
```

### 加粗、斜体、高亮

```
**加粗文本**

*斜体文本*

~~删除线文本~~

==高亮文本==

***加粗和斜体文本***
```

**加粗文本**

*斜体文本*

~~删除线文本~~

==高亮文本==

***加粗和斜体文本***

### 链接

```
[[Wiki 链接]]

[Markdown链接](xxx.md)


```
